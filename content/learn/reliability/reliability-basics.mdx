## 개요

신뢰성(Reliability)이란 **주어진 기능을 정해진 조건에서 의도한 기간 동안 고장 없이 수행할 확률**이다. 품질기술사 시험에서 신뢰성 문제는 수치 계산과 서술이 혼합되므로, 기본 함수 관계를 확실히 이해해야 한다.

---

## 신뢰성의 4대 정의 요소

| 요소 | 설명 | 예시 |
|------|------|------|
| **기능** (Function) | 요구되는 성능 또는 동작 | 정격 출력 유지 |
| **조건** (Conditions) | 환경·운용 조건 | 온도 −20~85°C, 정격 전압 |
| **기간** (Duration) | 임무 시간 또는 수명 | 10,000시간 |
| **확률** (Probability) | 수치로 표현된 신뢰도 | R = 0.95 (95%) |

---

## 신뢰도 함수와 고장 관련 함수

### 핵심 4개 함수의 관계

수명 확률변수 T에 대해 다음 4개 함수가 상호 유도된다.

```
신뢰도 함수   R(t) = P(T > t)               — 시간 t까지 생존할 확률
누적 고장분포 F(t) = 1 - R(t) = P(T ≤ t)   — 시간 t까지 고장날 확률
고장밀도 함수 f(t) = -dR(t)/dt = dF(t)/dt  — 순간 고장 확률밀도
고장률 함수   λ(t) = f(t) / R(t)            — 생존한 아이템의 순간 고장률
```

### R(t)와 λ(t)의 역방향 관계

고장률 함수로부터 신뢰도 함수를 구하는 관계:

```
R(t) = exp[ -∫₀ᵗ λ(u) du ]
```

λ(t)가 상수(λ)이면:

```
R(t) = exp(-λt)     — 지수분포 (CFR, 상수 고장률)
```

---

## 지수분포와 CFR(Constant Failure Rate)

지수분포는 고장률이 시간에 무관하게 일정한 경우 사용되며, 우발고장기(욕조곡선 중간 구간)에 적합하다.

| 함수 | 지수분포 표현 |
|------|--------------|
| 고장률 | λ(t) = λ (상수) |
| 고장밀도 | f(t) = λ·exp(-λt) |
| 신뢰도 | R(t) = exp(-λt) |
| 누적 고장분포 | F(t) = 1 - exp(-λt) |

> **무기억성(Memoryless Property)**: 지수분포는 `P(T > s+t | T > s) = P(T > t)`를 만족한다. 즉, 이미 s시간 동안 고장 없이 작동한 이후에도 앞으로 t시간 더 생존할 확률은 처음부터 t시간 생존할 확률과 동일하다. 이는 우발 고장을 모델링하는 데 중요한 특성이다.

## MTTF · MTBF · MTTR 비교

| 지표 | 영문 | 정의 | 적용 대상 | 공식 |
|------|------|------|----------|------|
| **MTTF** | Mean Time To Failure | 최초 고장까지 평균 시간 | 비수리 품목(소모품, 일회용) | `MTTF = ∫₀^∞ R(t) dt` |
| **MTBF** | Mean Time Between Failures | 고장 간 평균 시간 | 수리 가능 시스템 | `MTBF = 총 가동시간 ÷ 고장 건수` |
| **MTTR** | Mean Time To Repair | 평균 수리(복구) 시간 | 수리 가능 시스템 | `MTTR = 총 수리시간 ÷ 고장 건수` |

> 지수분포에서 **MTTF = 1/λ**, **MTBF ≈ MTTF** (수리 후 동일 분포 가정 시).

## 가용도 (Availability)

가용도는 시스템이 필요한 시점에 정상 작동할 확률이다.

### 고유 가용도 (Inherent Availability)

```
A = MTBF / (MTBF + MTTR)
```

### 가용도 종류

| 종류 | 고려 요소 | 용도 |
|------|----------|------|
| 고유 가용도 (Ai) | MTBF, MTTR만 포함 | 설계 단계 이론값 |
| 달성 가용도 (Aa) | 예방보전 시간 추가 포함 | 운용 계획 |
| 운용 가용도 (Ao) | 대기시간·행정지연 포함 | 실제 운용 환경 |

## 시험 예제

<ExamCase>
**[예제 1]** 어떤 전자 부품의 MTBF가 200시간이며 지수분포를 따른다고 한다.

**(1)** 고장률 λ를 구하라.
**(2)** t = 100시간에서의 신뢰도 R(100)을 구하라.
**(3)** R(t) = 0.5가 되는 시간(반감기)을 구하라.
</ExamCase>

<RevealAnswer>
**(1) 고장률 λ**
```
λ = 1 / MTBF = 1 / 200 = 0.005 (건/시간)
```

**(2) t = 100시간에서의 신뢰도**
```
R(100) = exp(-λ × t) = exp(-0.005 × 100) = exp(-0.5) ≈ 0.6065
```
→ 100시간 후 약 **60.65%** 신뢰도

**(3) 반감기 t₁/₂**
```
R(t) = 0.5 = exp(-0.005 × t)
-0.005t = ln(0.5) = -0.6931
t = 0.6931 / 0.005 ≈ 138.6시간
```
→ 약 **139시간** 후 신뢰도가 50% 이하로 떨어짐
</RevealAnswer>

---

<ExamCase>
**[예제 2]** 어느 설비가 720시간 동안 가동되면서 6번 고장났고, 총 수리시간은 24시간이었다.

**(1)** MTBF, MTTR을 계산하라.
**(2)** 고유 가용도(Ai)를 구하라.
**(3)** 고유 가용도를 99% 이상으로 개선하려면 MTTR을 몇 시간 이하로 줄여야 하는가?
</ExamCase>

<RevealAnswer>
**(1) MTBF와 MTTR**
```
총 가동시간 = 720 - 24 = 696시간
MTBF = 696 ÷ 6 = 116시간
MTTR = 24 ÷ 6 = 4시간
```

**(2) 고유 가용도**
```
Ai = MTBF / (MTBF + MTTR) = 116 / (116 + 4) = 116 / 120 ≈ 0.9667
```
→ 가용도 **96.67%**

**(3) MTTR 개선 목표**
```
0.99 ≤ 116 / (116 + MTTR)
116 + MTTR ≤ 116 / 0.99 = 117.17
MTTR ≤ 1.17시간 (약 70분)
```
→ 평균 수리시간을 **약 70분 이내**로 단축해야 함
</RevealAnswer>

---

<ExamCase>
**[예제 3 — 유도형]** 고장률이 λ인 지수분포를 따르는 부품의 MTTF = 1/λ임을 적분으로 유도하라.
</ExamCase>

<RevealAnswer>
신뢰도 함수 `R(t) = exp(-λt)`를 이용하여 MTTF를 적분으로 구한다:

```
MTTF = E[T] = ∫₀^∞ R(t) dt
            = ∫₀^∞ exp(-λt) dt
            = [-1/λ · exp(-λt)]₀^∞
            = (0) - (-1/λ · 1)
            = 1/λ
```

따라서 지수분포에서 **MTTF = 1/λ**.

예를 들어 λ = 0.005(건/시간)이면 MTTF = 200시간.
</RevealAnswer>

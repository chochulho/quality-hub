## ESG 경영이란?

ESG는 기업의 비재무적 성과를 측정하는 세 가지 축이다.

| 축 | 영문 | 핵심 내용 |
|----|------|---------|
| **E** | Environmental | 탄소 배출, 에너지·자원 사용, 환경영향 |
| **S** | Social | 안전보건, 인권, 공급망 윤리, 지역사회 |
| **G** | Governance | 이사회 구성, 반부패, 투명한 정보 공개 |

### 왜 지금 ESG가 중요한가?

- **공급망 압력**: 글로벌 OEM들이 협력사에 ESG 보고서·인증 요구 증가
- **금융 연계**: ESG 등급이 대출 금리·투자 유입에 직접 영향
- **규제 강화**: EU 공급망 실사법(CSDDD), CBAM(탄소국경조정) 등
- **IATF 연계**: IATF 16949의 기업 책임(5.1.1) 조항이 ESG와 직결

## 주요 ISO 인증 프로그램

### 전체 지형도

| ISO 표준 | 분야 | ESG 축 | 인증 여부 |
|---------|------|-------|---------|
| ISO 9001 | 품질경영 | G | 인증 가능 |
| IATF 16949 | 자동차 QMS | G | 인증 가능 |
| ISO 14001 | 환경경영 | E | 인증 가능 |
| ISO 45001 | 안전보건경영 | S | 인증 가능 |
| ISO 50001 | 에너지경영 | E | 인증 가능 |
| ISO 27001 | 정보보안 | G | 인증 가능 |
| ISO 26000 | 사회적 책임 | S + G | 가이던스 (인증 없음) |

### ISO 14001 — 환경경영시스템 (EMS)

**목적**: 환경영향을 체계적으로 파악하고 줄이기 위한 경영시스템

**핵심 요건**:

| 조항 | 요건 내용 |
|------|---------|
| 6.1 | **환경측면 파악**: 조직 활동 중 환경에 영향을 주는 요소 식별 (예: 폐수·소음·CO2) |
| 6.1 | **법적 요구사항**: 환경 관련 법령·규제 파악 및 준수 |
| 6.2 | 환경 목표 수립 (계량적 목표 + 달성 계획) |
| 8.1 | 환경측면 통제 (운용 기준 설정) |
| 9.1 | 환경 성과 모니터링·측정 |
| 9.3 | 경영검토에 환경 성과 포함 |

**제조업 주요 환경측면 예시**:
- 에너지 사용 (전기·가스)
- 폐기물 발생 (금속 스크랩·포장재)
- 폐수 배출
- 유해화학물질 사용 (절삭유·세척제)
- 소음·진동

---

### ISO 45001 — 안전보건경영시스템 (OH&SMS)

**목적**: 작업장 내 부상·질병 예방, 근로자 안전보건 보호

**핵심 요건**:

| 조항 | 요건 내용 |
|------|---------|
| 6.1 | **위험성 평가**: 위험요인 식별 → 리스크 평가 → 통제 방법 결정 |
| 6.1 | 법적 요구사항 (산업안전보건법 등) |
| 7.4 | 안전보건 의사소통 (근로자 참여 필수) |
| 8.1 | 위험 통제 계층 적용 (제거 → 대체 → 공학적 통제 → 행정적 통제 → PPE) |
| 9.1 | 사고·아차사고 조사 및 보고 |

> **OHSAS 18001과의 관계**: ISO 45001이 2018년 발행되면서 OHSAS 18001은 폐지됨. 기존 OHSAS 인증 조직은 ISO 45001로 전환 완료.

---

### ISO 50001 — 에너지경영시스템 (EnMS)

**목적**: 에너지 효율 향상, 에너지 비용 절감, 탄소 배출 감소

**핵심 개념**:

| 용어 | 설명 |
|------|------|
| SEU (중요 에너지 사용처) | 에너지 소비량이 크거나 개선 가능성이 높은 설비·공정 |
| EnB (에너지 기준선) | 성과 비교의 기준이 되는 기간의 에너지 소비량 |
| EnPI (에너지 성과지표) | 에너지 효율 측정 지표 (예: 생산량 당 kWh) |

---

### ISO 27001 — 정보보안경영시스템 (ISMS)

**목적**: 정보 자산의 기밀성·무결성·가용성 보호

**제조업 관련성**:
- 고객 도면·설계 데이터 보호
- 생산 시스템(MES·ERP) 보안
- 원격 접속·클라우드 사용 증가로 중요성 증대

## 통합경영시스템 (IMS: Integrated Management System)

여러 ISO 표준을 별도로 운영하면 심사 부담·문서 중복이 커진다. **HLS(High Level Structure)**를 이용해 통합 운영하면 효율이 크게 높아진다.

### HLS란?

ISO가 2012년부터 모든 경영시스템 표준에 적용한 공통 구조(조항 1~10). 이 덕분에 ISO 9001·14001·45001이 같은 뼈대를 공유한다.

```
공통 조항 (HLS)
├── 4. 조직 상황
├── 5. 리더십
├── 6. 기획
├── 7. 지원
├── 8. 운용
├── 9. 성과평가
└── 10. 개선
```

### 통합 운영 예시 (IATF + ISO 14001 + ISO 45001)

| 영역 | 통합 방법 |
|------|---------|
| 방침 | 품질·환경·안전보건 방침 하나의 문서로 통합 |
| 목표 | KPI 표에 품질·환경·안전 지표 함께 관리 |
| 내부심사 | 1회 심사에서 3개 표준 동시 심사 가능 |
| 경영검토 | 통합 경영검토 의제 (품질·환경·안전 성과 함께 보고) |
| 문서 관리 | 공통 문서관리 절차 1개로 통합 |

### IMS 구축 순서

```
1단계  기준 표준 선택 (보통 IATF 16949 또는 ISO 9001)
2단계  추가 표준 Gap 분석 (ISO 14001, ISO 45001 등)
3단계  통합 문서 체계 설계
4단계  통합 내부심사 프로그램 수립
5단계  단계별 인증 심사 (같은 CB에서 동시 심사 권장)
```

## 자동차 산업 공급망 ESG 요구사항

국내외 주요 OEM의 ESG 관련 공급망 요구사항이 강화되고 있다:

| 요구사항 | 내용 |
|---------|------|
| 탄소 배출 보고 | Scope 1·2·3 온실가스 배출량 제출 |
| RBA 준수 | 책임있는 비즈니스 연합 행동 규범 |
| 광물 추적 | 분쟁 광물(3TG) 미사용 보증 |
| 화학물질 관리 | IMDS(자동차 소재 데이터 시스템) 등록 |
| 환경 인증 | ISO 14001 인증 요구하는 고객 증가 |

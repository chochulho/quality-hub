## 공정능력지수란?

공정능력지수(Process Capability Index)는 공정이 규격(Specification)을 얼마나 잘 충족하는지를 나타내는 통계적 지표입니다. 품질기술사 시험에서 단골 출제 항목이며, 실무에서도 IATF 16949 조항 8.3.5(공정능력 검증)에서 요구합니다.

## Cp (공정능력지수)

Cp는 공정 변동 대비 규격 폭의 비율입니다.

```
Cp = (USL - LSL) / (6σ)
```

- **USL**: 규격 상한 (Upper Specification Limit)
- **LSL**: 규격 하한 (Lower Specification Limit)
- **σ**: 공정 표준편차

**Cp의 한계**: Cp는 공정 산포만 고려하고, 공정 중심이 규격 중심과 일치하는지는 반영하지 않습니다.

## Cpk (수정 공정능력지수)

Cpk는 공정 중심(평균)의 편향(Bias)을 함께 고려합니다.

```
Cpk = min(Cpu, Cpl)
Cpu = (USL - X̄) / (3σ)
Cpl = (X̄ - LSL) / (3σ)
```

Cpk는 항상 Cp보다 작거나 같습니다 (Cpk ≤ Cp).

## 해석 기준

| 값 | 해석 |
|---|---|
| Cpk &lt; 1.00 | 공정 불합격 — 즉시 개선 필요 |
| 1.00 ≤ Cpk &lt; 1.33 | 조건부 합격 — 주의 필요 |
| 1.33 ≤ Cpk &lt; 1.67 | 합격 (일반 기준) |
| Cpk ≥ 1.67 | 우수 (IATF 신규 공정 기준) |

**IATF 16949 기준**: 신규 공정은 Cpk ≥ 1.67, 양산 공정은 Cpk ≥ 1.33이 일반적인 고객 요구사항입니다.

## MSA란?

MSA(Measurement System Analysis, 측정 시스템 분석)는 **측정 결과에 포함된 변동이 실제 제품 변동인지, 측정 시스템 자체의 오류인지 구분**하는 분석입니다. IATF 16949 조항 **7.1.5.1 및 7.1.5.2**에서 수행을 명시적으로 요구합니다.

## 측정 변동의 구조

```
관측된 전체 변동
├── 실제 부품 변동 (Part-to-Part)
└── 측정 시스템 변동
    ├── 편의 (Bias)
    ├── 선형성 (Linearity)
    ├── 안정성 (Stability)
    └── Gage R&R
        ├── 반복성 (Repeatability, EV)
        └── 재현성 (Reproducibility, AV)
```

## 5가지 측정 변동 요인

| 요인 | 정의 | 주요 원인 |
|---|---|---|
| **편의 (Bias)** | 기준값 대비 측정 평균의 차이 | 영점 조정 불량, 게이지 마모 |
| **선형성 (Linearity)** | 측정 범위 전반에 걸친 편의 변화 | 게이지 설계 결함, 센서 비선형 |
| **안정성 (Stability)** | 시간에 따른 편의 변화 | 게이지 노후화, 온도 드리프트 |
| **반복성 (EV)** | 동일 측정자·부품에서의 반복 변동 | 게이지 정밀도 부족 |
| **재현성 (AV)** | 다른 측정자 간 변동 | 측정 방법·숙련도 차이 |

## Gage R&R 계산 및 판정

```
GRR  = √(EV² + AV²)
%GRR = (GRR / 전체 공정 변동) × 100
```

| %GRR | 판정 | 조치 |
|---|---|---|
| **&lt; 10%** | 우수 | 측정 시스템 승인 |
| **10 ~ 30%** | 조건부 허용 | 개선 권고 |
| **> 30%** | 불합격 | 측정 시스템 개선 필수 |

**교차(Crossed) vs 중첩(Nested) Gage R&R**: 부품을 재측정할 수 있으면 교차형, 파괴 검사처럼 재측정이 불가능하면 중첩형을 사용합니다. AIAG MSA 4판은 ANOVA 방식을 권장합니다.

## 교정성적서 해석 및 판정 기준

교정은 **측정기를 국가 측정 표준에 소급(Traceability)되는 기준기와 비교**하여 오차를 파악하는 절차입니다.

### 교정성적서 구성 요소

| 항목 | 내용 |
|---|---|
| **측정기 정보** | 기기명, 모델, 일련번호 |
| **기준기 정보** | 사용 표준기 및 소급 출처 |
| **교정 결과** | 지시값 / 기준값 / 오차(편의) |
| **측정불확도(U)** | 확장불확도 값 및 포함인수(k) |
| **판정** | 합격 / 불합격 / 조건부 |
| **유효기간** | 차기 교정 예정일 |

### 측정불확도(U) 읽는 법

```
U = k × u_c

k = 2  → 신뢰수준 약 95%  (일반적)
k = 3  → 신뢰수준 약 99.7%
```

"U = 0.5 µm (k=2)" 의미 → 측정값 ± 0.5 µm 범위 안에 참값이 95% 확률로 존재

### 합격 판정 기준

- **단순 비교**: `|측정 오차| ≤ 허용 오차` → 합격
- **En값(Normalized Error)**: 기관 간 비교 측정 시 사용

```
       |x_lab - x_ref|
En = ──────────────────────
      √(U_lab² + U_ref²)

En ≤ 1.0 → 합격
```

ILAC G8 가이드 기준으로 측정불확도가 허용 오차의 1/3 이하일 때 판정이 신뢰성 있습니다.

### KOLAS 인정 성적서 식별

KOLAS(한국인정기구) 인정 교정 기관 성적서에는 **KOLAS 인정 마크 + 등록 번호**가 표시됩니다. IATF 16949 심사 시 ISO/IEC 17025 인정 기관 여부를 확인합니다.

### 교정 주기 설정 기준

교정 주기는 법정 기준이 아닌 **위험 기반**으로 사내에서 결정합니다. 사용 빈도, 환경 조건(고온·진동), 이전 교정 이력, CC/SC 특성 여부를 고려하여 단축 여부를 판단합니다.

## 게이지 관리 범위와 검교정 요구사항

### 관리 대상 게이지의 범위

IATF 16949에서 관리 대상은 **제품 합격·불합격 판정에 영향을 미치는 모든 도구**를 포함합니다.

#### 일반 측정기
버니어캘리퍼스, 마이크로미터, CMM, 하이트게이지, 토크렌치, 온도계 등 수치 측정이 가능한 계측기.

#### 양불마스터 (Go/No-Go Master Gauge)

기능 게이지의 **영점 설정 및 유효성 확인**에 사용하는 기준 게이지.

- 치수 정보를 가지므로 **교정 대상 (소급성 필요)**
- 주기적 치수 재검 (마모 확인), 사용 후 방청 보관
- 게이지 대장에 등록하여 교정 이력 관리

#### 한도견본 (Limit Sample / Boundary Sample)

외관·감성 특성처럼 수치로 표현이 어려운 특성의 **합격/불합격 경계를 현물로 보여주는 샘플**.

- 수치 측정 불가 → **교정 불가** — 대신 **승인 이력 관리** 필요
- 고객 또는 내부 승인 문서 유지
- 유효기간 설정: 변색·마모·오염 등 물리적 열화 모니터링
- 폐기 시 대체 한도견본 재승인 절차 수행

#### 셋팅지그 (Setting Jig / Reference Gauge)

공정 내 측정기(공기 마이크로미터, 전용 게이지 등)의 **기준점(Zero Point)을 설정**하는 데 사용.

- 기준 치수를 가지므로 **교정 대상 (소급성 필요)**
- 셋팅지그의 오차는 공정 측정기 전체에 영향 → MSA에 포함 검토
- 교정 주기는 공정 측정기와 연동하여 설정

### IATF 16949 요구사항 요약

| 조항 | 내용 |
|---|---|
| **7.1.5.1** | 측정기 교정·검증, 국가 표준 소급성, 교정 기록 보관 |
| **7.1.5.2** | 특별 특성(CC/SC) 측정 시스템에 MSA 수행 |

외부 교정 기관은 **ISO/IEC 17025 인정 기관** 우선 사용. 교정 기간 초과 시 즉시 사용 중지 → 해당 게이지로 검사된 제품 소급 검토 → 재교정 후 합격 확인 순으로 조치합니다.

## 관리도란?

관리도(Control Chart)는 공정 데이터를 시간 순서대로 타점(plot)하고, **관리한계선(UCL/LCL)** 을 기준으로 공정의 안정 여부를 판단하는 SPC의 핵심 도구입니다.

```
목적: 공정의 이상 원인(Special Cause)을 조기에 탐지
     → 불량 발생 전에 공정을 제어
```

관리도는 두 가지 원인을 구분합니다.

| 구분 | 설명 | 관리 방법 |
|---|---|---|
| **우연 원인** (Common Cause) | 항상 존재하는 불가피한 변동 (재료 미세 편차 등) | 공정 재설계로만 제거 |
| **이상 원인** (Special Cause) | 비정상적 이벤트 (설비 마모, 작업자 교체 등) | 즉시 원인 추적 후 제거 |

관리도의 목표는 **이상 원인이 발생한 순간을 빠르게 감지**하는 것입니다. IATF 16949 9.1.1은 SPC를 통한 공정 모니터링을 요구합니다.

## 관리도 선택 기준

가장 중요한 분기는 **계량형(연속 데이터)** vs **계수형(이산 데이터)** 입니다.

```
측정값인가?
  ├─ YES → 계량형 관리도
  │    ├─ 군 크기 n ≥ 2 → X̄-R 또는 X̄-S
  │    └─ 군 크기 n = 1 → I-MR (개별값)
  └─ NO  → 계수형 관리도
       ├─ 불량 개수 (부적합품수) → p-chart / np-chart
       └─ 결점 수 (부적합수)   → c-chart / u-chart
```

### 언제 어떤 차트를 쓰나?

| 차트 | 적합한 상황 | 군 크기 |
|---|---|---|
| **X̄-R** | 가장 일반적. 생산 샘플링 | 2 ~ 10 |
| **X̄-S** | 군 크기가 크거나 변동. S(표준편차) 더 정확 | ≥ 10 |
| **I-MR** | 측정값 1개씩 (파괴검사, 화학 분석 등) | 1 |
| **p-chart** | 불량률 (lot 크기 가변) | 변동 |
| **np-chart** | 불량 개수 (lot 크기 고정) | 고정 |
| **c-chart** | 단위당 결점 수 (lot 고정) | 고정 |
| **u-chart** | 단위당 결점 수 (lot 가변) | 변동 |

## X̄-R 관리도 — 계산 방법

### 기본 통계량

- **X̄ᵢ**: i번째 군의 평균 = ΣXᵢⱼ / n
- **Rᵢ**: i번째 군의 범위 = X_max − X_min
- **X̄̄ (Grand Mean)**: 전체 군 평균 = ΣX̄ᵢ / k
- **R̄**: 군 범위 평균 = ΣRᵢ / k

(k = 군 수, n = 군 크기)

### X̄ 관리도 한계

```
UCL_X̄ = X̄̄ + A₂ × R̄
CL_X̄  = X̄̄
LCL_X̄ = X̄̄ − A₂ × R̄
```

### R 관리도 한계

```
UCL_R = D₄ × R̄
CL_R  = R̄
LCL_R = D₃ × R̄   (n ≤ 6이면 0으로 처리)
```

### 관리도 계수표 (부분)

| n | A₂ | D₃ | D₄ | d₂ |
|:---:|:---:|:---:|:---:|:---:|
| 2 | 1.880 | 0 | 3.267 | 1.128 |
| 3 | 1.023 | 0 | 2.574 | 1.693 |
| 4 | 0.729 | 0 | 2.282 | 2.059 |
| 5 | 0.577 | 0 | 2.114 | 2.326 |
| 6 | 0.483 | 0 | 2.004 | 2.534 |
| 7 | 0.419 | 0.076 | 1.924 | 2.704 |
| 8 | 0.373 | 0.136 | 1.864 | 2.847 |
| 9 | 0.337 | 0.184 | 1.816 | 2.970 |
| 10 | 0.308 | 0.223 | 1.777 | 3.078 |

> **σ 추정**: σ̂ = R̄ / d₂ (R̄ 기반 추정치, 관리한계 계산에 사용)

## I-MR 관리도 — 개별값-이동범위

군 크기 n = 1인 경우 개별값(Individual, I)과 이동범위(Moving Range, MR)를 따로 관리합니다.

### 이동범위 계산

```
MRᵢ = |Xᵢ − Xᵢ₋₁|    (i = 2, 3, ..., k)
MR̄  = ΣMRᵢ / (k − 1)
```

### I 관리도 한계

```
UCL_I = X̄ + 3 × (MR̄ / d₂)    (d₂ = 1.128, n=2 기준)
CL_I  = X̄
LCL_I = X̄ − 3 × (MR̄ / d₂)
```

### MR 관리도 한계

```
UCL_MR = D₄ × MR̄    (D₄ = 3.267, n=2 기준)
CL_MR  = MR̄
LCL_MR = 0
```

## 관리한계 vs 규격한계 — 혼동 주의

| 구분 | 기준 | 목적 |
|---|---|---|
| **관리한계** (UCL/LCL) | 데이터에서 계산 (3σ 기반) | 공정 안정성 판단 |
| **규격한계** (USL/LSL) | 고객/설계에서 결정 | 제품 합격 판단 |

관리도의 관리한계와 도면의 규격한계는 **별개의 개념**입니다. 관리도는 "공정이 일관되게 운전되고 있는가?"를 묻고, 공정능력(Cp/Cpk)은 "그 공정이 규격을 만족하는가?"를 묻습니다.

→ 공정능력지수 이론은 [공정능력지수 Cp/Cpk](/learn/capability/cp-cpk) 참조

## 이상 판정 규칙

공정이 관리 상태(in-control)인지를 판정하는 규칙으로, **Nelson 규칙(8개)** 과 **Western Electric 규칙(4개)** 이 대표적입니다.

관리한계 구역을 다음과 같이 정의합니다.
```
   ← 구역 A →← 구역 B →← 구역 C →← 구역 C →← 구역 B →← 구역 A →
  |    3σ    |    2σ    |    1σ    |   CL   |    1σ    |    2σ    |    3σ    |
 LCL                                                                       UCL
```

### Western Electric 규칙 (4개)

실무에서 가장 많이 사용되는 기본 규칙입니다.

| 규칙 | 설명 | 신호 의미 |
|---|---|---|
| **WE-1** | 1점이 3σ 밖 (UCL 또는 LCL 초과) | 극단적 이상 발생 |
| **WE-2** | 연속 3점 중 2점이 같은 방향 2σ 밖 (구역 A) | 중간 크기 이동 |
| **WE-3** | 연속 5점 중 4점이 같은 방향 1σ 밖 (구역 B 이상) | 작은 이동 지속 |
| **WE-4** | 연속 8점이 모두 중심선 같은 쪽 | 공정 평균 이동 |

### Nelson 규칙 (8개)

WE 규칙을 포함하고 추가 패턴을 감지합니다.

| 규칙 | 설명 |
|---|---|
| **N-1** | 1점이 3σ 밖 |
| **N-2** | 연속 9점이 중심선 같은 쪽 |
| **N-3** | 연속 6점이 단조 증가 또는 단조 감소 |
| **N-4** | 연속 14점이 교대로 상승·하강 (진동) |
| **N-5** | 연속 3점 중 2점이 같은 방향 2σ 밖 |
| **N-6** | 연속 5점 중 4점이 같은 방향 1σ 밖 |
| **N-7** | 연속 15점이 모두 1σ 이내 (구역 C만 분포) |
| **N-8** | 연속 8점이 1σ 밖에 분포 (구역 C 없음) |

> **N-7** (층별화): 데이터가 너무 집중된 경우 — 서로 다른 모집단에서 뽑았을 가능성 (예: 두 설비 혼합)  
> **N-8** (혼합): 데이터가 중심 근처에 없는 경우 — 두 공정 평균이 섞인 경우

## 공정 해석 흐름

```
1. R(또는 MR) 관리도부터 판단
   ├─ 이상 신호 있음 → 먼저 산포 이상 원인 제거
   └─ 관리 상태    → 다음 단계로

2. X̄(또는 I) 관리도 판단
   ├─ 이상 신호 있음 → 평균 이동/이상 원인 제거
   └─ 관리 상태    → 공정 안정 확인

3. 공정 안정 확인 후 → Cp/Cpk 계산
   ├─ Cpk ≥ 1.33 → 합격 (양산 단계 목표)
   ├─ 1.00 ≤ Cpk < 1.33 → 주의 (모니터링 강화)
   └─ Cpk < 1.00 → 공정 개선 필요
```

> **중요**: 공정이 불안정할 때 계산한 Cp/Cpk는 의미 없습니다. 반드시 관리도로 안정성을 먼저 확인하세요.

## 관리도 운용 실무 포인트

### 초기 관리한계 설정
1. 약 25군(또는 100개 데이터) 이상 수집
2. 이상점 제거 후 관리한계 재계산
3. 이상점 제거 시 반드시 원인 기록
4. 확정된 관리한계로 공정 모니터링 시작

### 관리한계 갱신 시점
- 공정이 의도적으로 변경된 경우 (4M 변경)
- 재료 변경, 설비 교체, 작업 방법 개정
- 충분한 새 데이터 축적 후 갱신

### IATF 16949 SPC 요구사항 (9.1.1)
- **관리 항목**: 특별 특성(Special Characteristic)에 SPC 의무 적용
- **관리한계**: 데이터 기반으로 설정, 추정 표준편차 사용
- **기록 보유**: 관리도, 이상 발생 기록, 시정 조치 기록
- **고객 요구**: 일부 고객은 Cpk 기준값 별도 명시 (예: Cpk ≥ 1.67)

## 자주 나오는 시험 문제 유형

**Q1.** X̄-R 관리도에서 n=5, X̄̄=50, R̄=8일 때 UCL_X̄와 UCL_R을 구하라.
- UCL_X̄ = 50 + 0.577 × 8 = **54.62**
- UCL_R = 2.114 × 8 = **16.91**

**Q2.** 다음 중 이상 원인으로 판정하는 것은?
- ① 연속 7점이 중심선 위  → **이상** (N-2 규칙 - 연속 9점의 완화 버전, WE-4는 8점)
- ② 1점이 2σ와 3σ 사이   → 정상 (1점만으로는 신호 아님)
- ③ 연속 6점 단조 감소    → **이상** (N-3 규칙)
- ④ 연속 4점이 1σ 이내   → 정상

**Q3.** I-MR 관리도를 사용하는 경우를 설명하라.
- 군 크기 n=1인 경우: 파괴검사, 화학 분석, 긴 생산 주기 공정 등 샘플링이 불가능하거나 측정 비용이 클 때 사용

## 관련 도구

공정이 안정적(관리 상태)임을 확인했다면 다음 단계는 공정능력 분석입니다.  
[공정능력 분석기](/capability)에서 LSL/USL과 측정 데이터를 입력하면 Cp·Cpk·Pp·Ppk를 즉시 계산하고 히스토그램과 함께 확인할 수 있습니다.

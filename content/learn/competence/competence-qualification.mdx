## 왜 헷갈리는가?

일상 언어에서는 "자격이 있다"와 "적격하다"를 같은 뜻으로 쓰지만,
ISO 9001 품질경영 체계에서는 **명확하게 다른 개념**입니다.

```
자격 (Qualification/Credential)
  → 공식 인증: 자격증, 면허, 학위, 수료증
  → 투입(Input) 지표: "이 교육을 받았다는 증거"
  → 예: 용접기능사 자격증 보유

적격성 (Competence)
  → 실제 수행 능력: 지식 + 기술 + 경험의 결합
  → 결과(Output) 지표: "실제로 그 업무를 제대로 수행할 수 있다"
  → 예: 해당 공정의 용접을 규격에 맞게 단독 수행 가능
```

**핵심 차이**: 자격증을 가졌다고 해서 반드시 적격성이 있는 것은 아닙니다.
반대로, 자격증 없이도 OJT와 경험을 통해 적격성을 갖출 수 있습니다.

## 1. ISO 9001 7.2 — 적격성 요구사항

ISO 9001 7.2는 "자격증 보유 여부"가 아니라 **"실제 업무 수행 능력"**을 요구합니다.

### 7.2 조항 4가지 의무

```
조직은:
a) 품질에 영향을 미치는 업무를 수행하는 인원에게
   필요한 적격성을 결정하고,

b) 해당 인원이 교육, 훈련 또는 경험에 근거해
   적격성을 갖추도록 보장하고,

c) 해당되는 경우 필요한 적격성 확보를 위한 조치를 취하고,
   취해진 조치의 효과성을 평가하고,

d) 적격성의 증거로서 적절한 문서화된 정보를 보유해야 한다.
```

### 조치의 예시 (c항)

적격성이 부족할 때 취할 수 있는 조치:

| 조치 유형 | 내용 |
|---|---|
| 교육·훈련 | 집합 교육, OJT, e-러닝 |
| 재배치 | 적격성 있는 다른 인원으로 교체 |
| 외부 채용 | 해당 적격성을 갖춘 인원 채용 |
| 외주 활용 | 해당 업무를 적격성 있는 외부 업체에 위탁 |

## 2. 자격과 적격성의 관계

| 구분 | 자격 (Qualification) | 적격성 (Competence) |
|---|---|---|
| **정의** | 공식적 인증 (자격증·면허·수료증) | 실제 업무 수행 능력 |
| **성격** | 투입(Input) | 결과(Output) |
| **판단 기준** | 문서 보유 여부 | 업무 수행 결과·관찰 |
| **취득 방법** | 시험·교육 이수 | 교육 + 훈련 + 경험 + 검증 |
| **ISO 요구** | 일부 업무에 법적 요구 | 7.2에서 모든 품질 관련 업무 요구 |
| **예시** | 용접기능사 자격증 | 해당 공정 용접을 규격에 맞게 수행 |

### 4가지 조합 사례

| 자격 보유 | 적격성 보유 | 상황 | 대응 |
|:---:|:---:|---|---|
| ✓ | ✓ | **이상적** — 자격과 능력 모두 갖춤 | 그대로 투입 |
| ✓ | ✗ | 자격증은 있지만 실력 없음 | OJT·현장 훈련 필요 |
| ✗ | ✓ | 자격증은 없지만 경험으로 능력 갖춤 | 내부 자격 인정 절차 |
| ✗ | ✗ | 둘 다 없음 | 투입 불가, 교육 후 재평가 |

> **법적 요구 자격이 있는 경우**: 자격증이 없으면 적격성과 관계없이 투입 불가.  
> 예: 크레인 운전원, 위험물 취급자, 특수 용접 공인 검사원 등.

## 3. 적격성 결정 4단계 프로세스

```
① 적격성 기준 결정 (업무별 요구 능력 정의)
        ↓
② 현재 적격성 평가 (다기능도·시험·관찰)
        ↓
③ GAP 분석 → 교육훈련 계획 수립
        ↓
④ 교육훈련 실시 → 효과성 평가
        ↓
    ← 부족 시 재교육 ←
        ↓ 충족 시
    적격성 확인·기록 보유
```

### ① 업무별 적격성 기준 정의

업무 분석을 통해 각 공정에 필요한 능력을 정의합니다.

| 공정 | 필요 지식 | 필요 기술 | 경험 기준 |
|---|---|---|---|
| 용접 | 용접 규격, 검사 기준 | 아크 용접 실기 | OJT 100시간 이상 |
| 외관 검사 | 불량 유형, 판정 기준 | 육안 검사 | OJT 40시간 + 통과 시험 |
| 프레스 | 설비 구조, 안전 규정 | 금형 교체, 설비 조작 | OJT 60시간 이상 |

### ④ 교육훈련 효과성 평가 방법

교육 후 단순히 "이수" 기록만으로는 부족합니다. **실제로 효과가 있었는지** 평가해야 합니다.

| 평가 방법 | 내용 | 시기 |
|---|---|---|
| **필기 시험** | 지식 확인 | 교육 직후 |
| **실기 평가** | 실제 업무 수행 관찰 | 교육 후 1~4주 |
| **OJT 관찰** | 현장 감독자 평가 | 현장 복귀 후 |
| **불량률 모니터링** | 교육 전후 불량률 비교 | 3개월 후 |
| **다기능도 갱신** | 레벨 상향 기록 | 평가 통과 후 |

> **IATF 16949 요구**: 효과성 평가 결과도 기록으로 보유해야 합니다.

## 4. IATF 16949 추가 요구사항

ISO 9001 7.2 외에 자동차 산업은 추가 적격성 요구가 있습니다.

| 조항 | 대상 | 내용 |
|---|---|---|
| **7.2.2** | 현장·보전 인원 | OJT 기록, 제품·공정 품질 교육 이력 |
| **7.2.3** | 내부 심사원 | IATF 16949 요건 교육 + 심사 경험 증빙 |
| **7.2.4** | 2자 심사원 | 7.2.3 요건 + VDA 6.3 또는 동등 자격 |

### 7.2.2 — 현장 인원 적격성 특수 요구

IATF 16949는 ISO 9001보다 훨씬 구체적인 기록을 요구합니다:

- 각 작업자의 **공정별 OJT 훈련 기록**
- **신규·전환 배치 시** 공정 승인(인가) 절차
- **다기능도(Skill Matrix)** 를 통한 현황 가시화
- 적격성이 없는 인원의 **감독 조건 명시**

## 5. 다기능도 (Skill Matrix)와의 연계

다기능도는 **팀 전체의 적격성 현황을 한눈에 파악**하기 위한 시각적 도구입니다.
적격성 관리 4단계 중 ②번(현황 평가)과 ③번(GAP 분석)을 담당합니다.

### 다기능도 구조

```
          [공정A] [공정B] [공정C] [공정D] [공정E] │ 인원별현황
공정별목표    3      2      3      2      3    │
─────────────────────────────────────────────────┤
김철수   │  ★      ●      △      ●      ○    │   3 / 3 ✓
이영희   │  ●      ○      ○      ★      ●    │   3 / 3 ✓
박민준   │  ◑      ★      ●      ○      ○    │   2 / 2 ✓
최지원   │  ○      ◑      ●      ○      △    │   1 / 3 ✗
─────────────────────────────────────────────────┤
공정별현황   3      2      2      2      1
목표대비     ✓      ✓      ✗      ✓      ✗
```

### 스킬 레벨 4단계 (표준 표기)

| 레벨 | 기호 | 의미 | 적격성 판정 |
|:---:|:---:|---|:---:|
| 0 | ○ | 교육 없음 / 수행 불가 | ✗ 부적격 |
| 1 | △ | 교육 중 (감독하 수행) | ✗ 부적격 |
| 2 | ◑ | 보조 가능 (지원 필요) | △ 조건부 |
| 3 | ● | 단독 수행 가능 | ✓ 적격 |
| 4 | ★ | 타인 지도 가능 (강사 역할) | ✓ 적격 + 교육 담당 |

> **적격성 기준선**: 일반적으로 **레벨 3 이상**을 해당 공정 적격자로 인정합니다.

### 공정별 목표와 인원별 목표

| 목표 유형 | 의미 | 설정 근거 |
|---|---|---|
| **공정별 목표** | 해당 공정에 최소 몇 명이 적격이어야 하나 | 리스크 평가: 1명이면 결근 시 생산 불가 |
| **인원별 목표** | 각 인원이 몇 개 공정을 수행할 수 있어야 하나 | 다기능화 수준: 유연 생산 대응 능력 |

두 목표를 동시에 관리해야 **공정 위험**과 **인원 유연성** 모두 확보됩니다.

### 다기능도 → 교육훈련 계획 연결

```
다기능도 GAP 분석
    ↓
공정별 부족 인원 확인 (목표 미달 공정)
    ↓
인원별 교육 필요 공정 목록 작성
    ↓
교육훈련 계획서 작성 (OJT·집합교육 일정)
    ↓
교육 실시 → 효과성 평가 → 다기능도 레벨 업데이트
```

## 6. 문서화 요구사항 (보유해야 할 기록)

IATF 16949 심사 시 요구되는 기록 목록:

| 기록 | 내용 | 보유 기간 |
|---|---|---|
| 다기능도 (Skill Matrix) | 현재 레벨 및 목표, 갱신 이력 | 최신 버전 + 1년 이전 |
| 교육훈련 계획서 | 연간 교육 계획 | 해당 연도 |
| 교육 이수 기록 | 일자, 과목, 시간, 강사, 참석자 서명 | 3년 이상 |
| 효과성 평가 결과 | 시험 점수, 현장 관찰 평가 | 교육 이력과 함께 |
| 적격성 인정서 | 공정별 수행 가능 인원 명단, 승인자 서명 | 유효 기간 동안 |

## 품질기술사 시험 출제 포인트

### 자주 나오는 문제 유형

- **적격성과 자격의 차이를 ISO 9001 7.2를 기준으로 설명하시오**
- **ISO 9001 7.2의 4가지 의무사항을 설명하시오**
- **다기능도의 구성 요소와 활용 목적을 설명하시오**
- **교육훈련 효과성 평가 방법 3가지 이상을 설명하시오**
- **공정별 목표와 인원별 목표의 차이를 설명하시오**

### 암기 키워드

```
자격 = 공식 인증(투입), 적격성 = 실제 능력(결과)
ISO 7.2 4가지: 결정 → 보장 → 조치+효과성평가 → 기록보유
레벨 0~4: 없음·교육중·보조·단독·지도가능
적격 기준선 = 레벨 3 이상 (단독 수행 가능)
공정별목표 = 리스크 관리 / 인원별목표 = 다기능화 수준
효과성 평가 = 필기시험 + 실기관찰 + 불량률 모니터링
IATF 7.2.2 = OJT 기록 + 신규배치 공정승인 + 다기능도
```

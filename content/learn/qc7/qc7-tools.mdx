## QC 7가지 도구란?

QC 7가지 도구(QC Seven Tools)는 **현장 품질 데이터를 수집·분석하여 문제를 해결하는 7가지 기본 도구**입니다. 1960년대 일본 TQC(전사적 품질관리) 운동에서 정립되었으며, 주로 **정량적(계수형·계량형) 데이터**를 다룹니다.

## 7가지 도구 한눈에 보기

| # | 도구 | 목적 | 사용 시점 |
|---|---|---|---|
| 1 | **체크시트** | 데이터 수집·집계 | 문제 발생 현황 파악 |
| 2 | **히스토그램** | 데이터 분포 파악 | 공정 산포 확인 |
| 3 | **파레토 차트** | 우선순위 결정 | 중요한 소수 문제 선별 |
| 4 | **특성요인도** | 원인 분석 | 근본 원인 탐색 |
| 5 | **산포도** | 변수 간 상관관계 | 원인-결과 관계 확인 |
| 6 | **층별** | 데이터 분리 | 특정 원인 그룹 파악 |
| 7 | **관리도** | 공정 모니터링 | 이상 원인 실시간 탐지 |

## ① 체크시트 (Check Sheet)

체크시트는 **데이터를 정해진 양식에 체계적으로 기록**하는 도구입니다. 조사 목적에 맞는 항목을 미리 정하고, 현장에서 발생할 때마다 체크(✓)로 기록합니다.

- **불량 유형별 빈도** 파악에 가장 많이 사용
- 이후 파레토 차트, 히스토그램의 원시 데이터가 됨
- 기록자가 판단 없이 단순 집계할 수 있어야 함

## ② 히스토그램 (Histogram)

히스토그램은 **연속 데이터를 구간(Bin)으로 나누어 빈도를 막대그래프로 표시**합니다.

- 공정 산포의 모양(정규·치우침·이봉·절단분포) 파악
- 규격 한계(USL/LSL)와 겹쳐 보면 공정능력 직관적 확인
- **계급 수 공식**: Sturges — `k ≈ 1 + 3.3 × log₁₀(n)`

**분포 형태별 의미**:
```
정규분포 → 공정 안정
우측 치우침(양성 왜도) → 하한 규격 쪽 불량 위험
이봉분포 → 두 개 모집단 혼재 (층별 필요)
```

## ③ 파레토 차트 (Pareto Chart)

파레토 차트는 **항목을 빈도 내림차순으로 정렬한 막대그래프 + 누적비율 꺾은선**의 복합 차트입니다.

- **80:20 법칙**: 전체 불량의 80%는 20%의 원인에서 발생
- 80% 기준선 왼쪽 항목 = 집중 개선 대상
- "기타"는 빈도와 무관하게 항상 맨 오른쪽에 배치

## ④ 특성요인도 (Fishbone / Ishikawa)

특성요인도는 **결과(불량)에 영향을 미치는 원인을 4M+1E 체계로 분류**한 어골(魚骨) 형태의 다이어그램입니다.

| 카테고리 | 내용 |
|---|---|
| **Man** | 작업자 기술, 훈련, 경험 |
| **Machine** | 설비, 공구, 치공구 마모·정밀도 |
| **Material** | 원부자재 품질, 공급업체 |
| **Method** | 작업 표준, 조건 설정 |
| **Environment** | 온도, 습도, 조명, 분진 |

파레토로 1순위 불량을 선정 → 특성요인도로 원인 분석 → 5-Why로 근본 원인 도출 순서로 연계합니다.

## ⑤ 산포도 (Scatter Diagram)

산포도는 **두 변수(X, Y)의 관계를 좌표 평면의 점으로 시각화**합니다. Pearson 상관계수(r)로 관계 강도를 정량화합니다.

```
|r| ≥ 0.7  → 강한 상관
|r| ≥ 0.9  → 매우 강한 상관
|r| < 0.3  → 상관 없음
```

**주의**: 상관관계 ≠ 인과관계. 제3의 변수(교란 변수) 가능성 항상 고려.

## ⑥ 층별 (Stratification)

층별은 **혼재된 데이터를 의미 있는 소그룹(설비·작업자·원자재·시간)으로 나누어 비교**하는 방법입니다. 단독 도구라기보다 **다른 도구를 적용할 때의 방법론**입니다.

- 히스토그램이 이봉분포를 보일 때 → 층별 후 각각 그리기
- 관리도에서 간헐적 이상 패턴 → 교대별·설비별로 분리
- 전체 불량률은 낮은데 특정 설비에 집중될 때

## ⑦ 관리도 (Control Chart)

관리도는 **공정 데이터를 시간 순서로 플롯하고 통계적 관리 한계선(UCL/LCL)**으로 공정 이상을 탐지합니다. 월터 슈하트(Shewhart)가 1920년대 개발.

**관리도 종류 선택**:
```
계량형 데이터
├── 부분군 n ≥ 2 → X̄-R (n ≤ 10) / X̄-S (n > 10)
└── 부분군 n = 1 → I-MR 관리도

계수형 데이터
├── 불량 수     → np (일정) / p (가변)
└── 결점 수     → c (일정) / u (가변)
```

관리 한계(UCL/LCL)는 공정 자체 능력에서 계산 (X̄ ± 3σ). 규격 한계(USL/LSL)와 별개입니다.

## IATF 16949 연관 조항

- **9.1.1**: 모니터링·측정·분석·평가
- **9.1.1.1**: 통계적 도구 — 관리도를 포함한 SPC 적용 요구
- **10.2.1**: 부적합 및 시정 조치 — 특성요인도·파레토 활용

# 수입검사 지침

**문서번호**: QI-P01  **버전**: {{rev_no}}  **발행일**: {{created_at}}

**회사명**: {{company_name}}  **담당**: {{dept_quality}}

---

## 1. 목적

입고 자재·부품의 품질을 검사하여 부적합품이 생산 공정에 투입되는 것을 방지한다.

## 2. 적용 범위

원자재, 구매 부품, 외주 가공품 등 모든 외부 입고 품목에 적용한다.

## 3. 준비물

- 해당 품목 수입검사 기준서 (QF-S02-01)
- 교정된 측정 장비 (버니어 캘리퍼스, 마이크로미터 등)
- 수입검사 기록지 (QF-S02-02)
- 합격·불합격 태그 (초록/빨강)
- AQL 샘플링표 (ANSI/ASQ Z1.4)

---

## 4. 검사 절차

### Step 1: 입고 확인 (5분)

☐ 납품서와 발주서의 품번·수량·개정 번호 일치 확인  
☐ 포장 외관 이상 여부 확인 (파손, 오염, 라벨 탈락)  
☐ 불일치 발견 시 → {{dept_purchasing}}에 즉시 연락 후 검사 보류

### Step 2: 샘플 채취

☐ AQL 기준에 따라 샘플 수 결정:

| Lot 수량 | 검사 수준 II, AQL 1.0 | AQL 2.5 |
|--------|---------------------|---------|
| 2~8 | 전수 | 전수 |
| 9~150 | 13 | 8 |
| 151~500 | 50 | 32 |
| 501~1200 | 80 | 50 |
| 1201~3200 | 125 | 80 |

☐ 샘플은 Lot에서 무작위 채취  
☐ 특별특성(SC) 항목은 강화 샘플링 또는 전수 검사

### Step 3: 외관 검사

☐ 긁힘, 버, 크랙, 변형, 오염 확인  
☐ 도금·도장 품목: 균일성, 벗겨짐, 변색 확인  
☐ 불합격 기준: 기준서의 외관 불합격 정의에 해당하는 경우

### Step 4: 치수 검사

☐ 검사 기준서에 명시된 치수 항목 순서대로 측정  
☐ 특별특성은 반드시 포함  
☐ 측정값을 수입검사 기록지에 기재  
☐ 합격 판정: 전체 측정값이 허용 공차 내 → AQL 기준 합격 수 이하

### Step 5: 재질·성능 확인

☐ 공급자 시험 성적서 접수 및 기준 충족 여부 확인  
☐ 필요 시 사내 시험 실시 (인장, 경도, 기능 테스트)

### Step 6: 판정 및 기록

☐ **합격**: 초록 태그 부착 + 수입검사 기록지에 "합격" 기재 + 입고 스탬프  
☐ **불합격**: 빨강 태그 부착 + 불합격 사유 기재 + 격리 구역 이동  
☐ **특채 처리**: 노랑 태그 + 특채 번호 기재 + {{dept_quality}} 팀장 승인 필수

---

## 5. 불합격 처리 흐름

```
불합격 확인
  → 격리 구역 이동 (빨강 태그)
  → QF-S02-02 불합격 보고서 작성
  → {{dept_purchasing}} 에 통보 (1일 이내)
  → {{dept_purchasing}} → 공급자 반품 요청 또는 8D 요청
```

---

## 6. 주의사항

⚠️ 검사 완료 전 자재를 생산 공정에 투입하지 않는다.  
⚠️ 측정 장비는 사용 전 교정 유효 기간을 확인한다.  
⚠️ 긴급 입고의 경우에도 {{dept_quality}} 팀장 승인 없이 검사 생략 불가.

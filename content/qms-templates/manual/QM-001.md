## 1. 목적 및 적용 범위

### 1.1 목적

본 매뉴얼은 {{company_name}}(이하 "당사")의 품질경영시스템(Quality Management System, QMS)을 {{cert_target}} 요건에 적합하게 수립·문서화·유지·지속적으로 개선하기 위하여 작성되었습니다.

본 문서는 다음을 목적으로 합니다:

- QMS의 전반적인 구조, 적용 범위, 책임을 명확히 정의
- 품질방침 및 품질목표를 통해 조직의 전략적 방향을 제시
- 각 절차서(QP-xxx), 지침서(QI-xxx), 기록 양식과의 연계 구조 제공
- 고객, 인증기관, 이해관계자에게 QMS의 효과성을 증명

본 매뉴얼은 {{company_name}}의 **{{industry}}** 관련 사업 전반에 적용됩니다.

**포함되는 활동:**

## 3. 용어 및 정의

본 매뉴얼에서 사용하는 주요 용어는 ISO 9000:2015 및 IATF 16949:2016에 정의된 바에 따릅니다. 주요 용어를 다음과 같이 정리합니다:

| 용어 | 정의 |
|------|------|
| QMS | Quality Management System. 품질 관련 목표 달성을 지도·관리하는 경영시스템. |
| 품질방침 | 최고경영자가 공식 표명한 조직의 품질 관련 의도와 방향. |
| 품질목표 | 품질과 관련하여 추구하는 측정 가능한 성과 목표 (KPI). |
| 프로세스 | 입력을 출력으로 변환하는 상호 관련되거나 상호 작용하는 활동의 집합. |
| 절차서 (QP) | 활동 또는 프로세스 수행 방법을 규정한 문서. |
| 지침서 (QI) | 작업자 수준의 세부 작업 방법을 기술한 문서. |
| 기록 (FR) | 달성된 결과나 수행된 활동에 대한 증거를 제공하는 문서. |
| 부적합 | 요구사항을 충족하지 않는 상태. |
| 시정조치 (CA) | 부적합의 원인을 제거하고 재발을 방지하는 조치. |
| 리스크 | 불확실성의 영향. 목표에 대한 긍정적 또는 부정적 편차 가능성. |
| CSR | Customer Specific Requirements. 고객별 특수 요건. |
| APQP | Advanced Product Quality Planning. 신제품 품질계획. |
| PPAP | Production Part Approval Process. 양산 부품 승인 절차. |
| FMEA | Failure Mode and Effects Analysis. 잠재 고장형태 및 영향 분석. |
| Control Plan | 제품·공정 특성의 관리 방법을 정의한 문서. |
| MSA | Measurement System Analysis. 측정시스템 분석. |
| Cpk | 공정능력지수. 규격 중심과의 편차를 고려한 공정 능력 지표. |
| NCR | Non-Conformance Report. 부적합 보고서. |
| OEM | Original Equipment Manufacturer. 완성차 제조사 (IATF 적용 시). |

## 4. 조직 상황

### 4.1 내·외부 이슈 파악 (ISO 9001 / IATF 4.1)

{{company_name}}은 QMS의 목적과 전략적 방향에 영향을 미칠 수 있는 내·외부 이슈를 주기적으로 파악하고 검토합니다.

**이슈 파악 방법:**
- 연간 경영 전략 수립 시 SWOT 분석 실시
- 반기 경영검토(QP-MR01) 시 이슈 현황 갱신
- 고객 포털, 품질 클레임, 공급망 동향 등 외부 정보 수집

**주요 내부 이슈 분류:**

| 분류 | 이슈 예시 | 갱신 주기 |
|------|---------|---------|
| 인적 자원 | 핵심 인력 이탈 리스크, 역량 갭 | 연간 |
| 설비·기술 | 노후 설비 교체 계획, 신기술 도입 | 연간 |
| 재무 | 품질 비용(COQ) 추이, 투자 여력 | 반기 |
| 조직 문화 | 품질 의식 수준, 변화 관리 | 연간 |

**주요 외부 이슈 분류:**

| 분류 | 이슈 예시 | 갱신 주기 |
|------|---------|---------|
| 고객·시장 | 고객 요건 변화, 경쟁 심화 | 반기 |
| 규제·법규 | 환경 규제, 제품 안전 법규 변경 | 연간 |
| 공급망 | 원자재 수급 불안정, 공급자 품질 | 반기 |
| 기술 트렌드 | 전기차 전환, 디지털화 요구 | 연간 |

이슈 파악 결과는 경영검토 회의록(QP-MR01 서식)에 문서화하며, 리스크·기회 분석(6.1절)과 연계합니다.

#### 4.4.3 프로세스 관리 기준

각 프로세스는 다음 요소를 포함하여 관리됩니다:

| 관리 요소 | 설명 |
|---------|------|

## 1. 목적

측정·시험에 사용하는 계측기의 교정, 식별, 유지·보관 및 기록을 관리하여 측정 결과의 신뢰성과 소급 가능성(Traceability)을 확보하고, {{cert_target}} 요건을 충족한다.

## 2. 적용 범위

{{company_name}}에서 제품 적합성 판정에 사용하는 모든 측정·검사 장비에 적용한다. 버니어 캘리퍼스, 마이크로미터, 토크렌치, 온도계, 경도계, 3차원 측정기(CMM) 등을 포함하며, 단순 지시용(기록 불필요) 장비는 제외한다.

## 3. 책임과 권한

| 역할 | 책임 | 권한 |
|------|------|------|

| 각 부서장 | 소관 계측기 사용·보관 책임, 손상·분실 즉시 보고 | 부서 내 계측기 대출·반납 관리 |
| 외부 교정기관 | 공인 교정 실시, 교정 성적서 발급 (KOLAS 인정기관) | — |

| **INPUT** | 계측기 대장(QF-MS01-01), 교정 주기표, 외부 교정 성적서, MSA 결과 기록 |

## 5. 세부 절차

### P1. 계측기 등록 및 식별 — {{dept_quality}} (도입 즉시)

1. **계측기 대장 등록** (QF-MS01-01): 고유 관리번호, 기기명, 제조사·모델, 측정 범위, 최소 눈금(분해능), 정밀도(교정 허용 오차), 교정 주기, 담당 부서 입력
2. **고유 번호 부착**: 모든 계측기에 고유 관리번호 스티커 부착 (훼손 방지 라미네이팅 처리 권장)
3. **합격 태그 부착**: 최초 교정 완료 후 파란 합격 태그 부착 (교정일·유효 기간·다음 교정 예정일 표기)
4. **보관 조건 정의**: 온도·습도·진동 등 보관 환경 기준을 대장에 명시, 부적절한 보관 시 재교정 실시

### P2. 교정 계획 수립 — {{dept_quality}} (연간, 매년 12월)

1. **교정 방법 분류**:

| 구분 | 기준 | 비고 |
|------|------|------|
| 외부 교정 | KOLAS 인정기관 의뢰 | 계측 소급성 확보 필수 항목 |
| 내부 교정 | 상위 교정된 기준기 대비 자체 교정 | 절차서(QI-M01) 보유 필요 |

2. **교정 주기 결정**: 기기 유형, 사용 빈도, 사용 환경, 고객 CSR에 따라 결정 (일반적으로 12개월, 고정밀 장비는 6개월)
3. **연간 교정 계획표 작성**: 기기별 교정 예정월, 교정 방법(내부/외부), 담당자, 예산 포함
4. **MSA 계획 수립**: 특별 특성(SC/CC) 측정에 사용하는 계측기에 대해 Gage R&R, 직선성, 안정성 평가 일정 수립

{{#if cert.iatf}}
> **IATF 16949 §7.1.5.1.1**: 고객 특별 특성 측정에 사용하는 계측기는 MSA(측정 시스템 분석)를 실시하고, %GR&R ≤ 10%(우수), ≤ 30%(조건부 사용) 기준을 적용한다. MSA 결과는 고객 요구 시 제출한다.
{{/if}}

### P3. 교정 실시 — {{dept_quality}} + 외부 교정기관 (계획에 따라)

1. **외부 교정**:
   - KOLAS 인정기관에 의뢰 → 공인 교정 성적서 수령
   - 성적서 상 불확도(Uncertainty) 기재 여부 및 계측 소급성 확인
2. **내부 교정**:
   - 교정 절차서(QI-M01)에 따라 기준기 대비 교정 실시
   - 교정 기록(QF-MS01-02)에 측정값, 허용 오차 대비 결과, 판정(합격/불합격) 기재
3. **교정 후 처리**:
   - 합격: 파란 합격 태그 교체 (교정일·유효 기간 갱신)
   - 불합격: P4 절차에 따라 처리
4. **교정 유예 관리**: 교정 기한 초과 계측기는 즉시 사용 중단 및 황색 유예 태그 부착 → 교정 완료 전 사용 금지

### P4. 교정 불합격 처리 — {{dept_quality}} (불합격 확인 즉시)

1. **즉시 격리**: 불합격 판정 즉시 빨간 불합격 태그 부착 → 사용 중단 및 격리 보관
2. **소급 검토 실시**: 최근 정상 교정 시점 이후 해당 계측기로 측정·판정한 제품의 품질 영향 평가 (QF-MS01-04 소급 검토 보고서 작성)
3. **소급 검토 결과에 따른 조치**:
   - 영향 없음 확인: 기록 유지 후 종결
   - 영향 있음(제품 부적합 우려): QP-CA01 부적합 처리 연계 → 해당 제품 전수 재검사 또는 고객 통보 검토
4. **수리·대체**: 계측기 수리 후 재교정 → 합격 시 복귀 / 수리 불가 시 폐기 처리 및 대장 갱신

{{#if cert.iatf}}
> **IATF 16949 §7.1.5.1.1**: 교정 불합격 발생 시 소급 가능성(Retrospective Assessment) 평가는 의무이며, 평가 기록과 조치 내용을 문서화하여 보관한다.
{{/if}}

### P5. 기록 관리 — {{dept_quality}} (지속)

1. **교정 기록 보관**: 교정 성적서 및 내부 교정 기록 최소 2년 보관 (외부 교정 성적서는 3년 권장)
2. **전자 기록 백업**: 교정 대장 및 성적서 전자파일 정기 백업 (월 1회)
3. **MSA 기록 보관**: Gage R&R 등 MSA 기록은 해당 측정 시스템 사용 기간 + 2년 이상 보관
4. **기록 접근 통제**: 교정 기록 수정 시 이전 기록 유지(이력 추적 가능) 원칙 준수

## 1. 목적

측정 장비의 교정 상태를 관리하여 측정 결과의 정확성과 추적 가능성을 보장하고, 고객 요구사항 및 {{cert_target}} 요건을 충족한다.

## 2. 적용 범위

제품 적합성 판정에 사용하는 모든 측정·검사 장비 (버니어 캘리퍼스, 마이크로미터, 토크렌치, 온도계 등).

## 3. 책임과 권한

| 역할 | 책임 |
|------|------|
| {{dept_quality}} | 계측기 대장 관리, 교정 계획 수립·이행, 교정 불합격 처리 |
| 각 부서장 | 소관 계측기 사용·보관 책임 |

## 4. 절차

### 4.1 계측기 등록

1. 계측기 대장(QF-MS01-01)에 등록: 관리번호, 기기명, 규격, 교정 주기, 담당 부서
2. 각 계측기에 **고유 관리번호** 부착

### 4.2 교정 계획 수립

- 교정 주기: 기기 유형, 사용 환경, 고객 CSR에 따라 결정 (일반적으로 12개월)
- 연간 교정 계획표를 1월 말까지 작성

### 4.3 교정 실시

| 교정 방법 | 기준 |
|---------|------|
| 외부 교정 | KOLAS 인정 교정기관 (성적서 발급) |
| 내부 교정 | 상위 교정된 기준 장비로 교정 후 기록 |

- 외부 교정 성적서: **3년 보관**
- 교정 합격 시 → **파란 스티커** (교정일·유효 기간 표기)
- 교정 불합격 시 → 즉시 격리, 이전 측정 결과 유효성 검토

### 4.4 MSA (Measurement System Analysis)

- 특별특성(Special Characteristic) 측정 계측기는 **GR&R(Gage R&R)** 실시
- 합격 기준: %GR&R ≤ 10% (우수), ≤ 30% (조건부 사용)
- 고객 요구 시 MSA 결과 고객 제출

### 4.5 분실·손상 처리

1. 계측기 분실·손상 즉시 {{dept_quality}}에 보고
2. 해당 계측기로 측정한 이전 결과의 품질 영향 평가
3. 필요 시 부적합 처리(QP-CA01) 연계

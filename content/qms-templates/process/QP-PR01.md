# 생산계획 관리 프로세스

**문서번호**: QP-PR01  **버전**: {{rev_no}}  **발행일**: {{created_at}}
**회사명**: {{company_name}}  **프로세스 오너**: {{dept_production}}

---

## 1. 목적

고객 수요에 맞는 생산 계획을 수립·조정하여 납기를 준수하고, 재공·재고를 최적화하며,
생산 자원(인원·설비·자재)의 효율적 배분을 통해 경영 목표를 달성한다.

{{#if cert.iatf}}
> **IATF 16949 연계**: 이 프로세스는 IATF 16949:2016 §8.1(운영 기획 및 관리),
> §8.2.3(제품·서비스 요구사항 검토), §8.5.1(생산·서비스 제공 관리)와 연계된다.
{{/if}}

---

## 2. 적용 범위

{{company_name}}의 **모든 생산 품목**에 대한 수요 파악, 생산 계획 수립, 자재 소요 계획(MRP),
작업 지시 발행, 진도 관리, 실적 분석에 적용한다.
외주 가공 품목의 일정 조율도 포함한다.

## 5. 세부 절차

### P1. 고객 수요 파악 — {{dept_sales}} → {{dept_production}} (매주 월요일 오전)

1. {{dept_sales}}는 고객 주문서, 수요 예측(Forecast), EDI 데이터를 취합하여 {{dept_production}}에 제공한다.
2. 납기별 우선순위를 결정한다: **전략 고객 > 납기 임박 > 수익성** 순 적용.
3. 재고·재공 현황을 확인하여 순 생산 필요 수량을 산출한다.
4. 납기 충돌 시 {{dept_sales}}와 협의 후 고객에게 일정을 조율한다.

{{#if cert.iatf}}
> **IATF 요건**: 고객 생산 일정 변동(Engineering Change, 수량 변경 등) 발생 시
> **즉시 생산 계획에 반영**하고 QF-PR01-04(계획 변경 이력서)에 사유를 기록한다.
{{/if}}

---

### P2. 생산 계획 수립 — {{dept_production}} (주간: 매주 금요일 / 월간: 전월 말일)

1. **생산 가능 수량** 산식:
   ```
   가용 생산 수량 = 설비 가용 시간 × 시간당 생산 속도 × 인원 투입률
   ```
2. Bottleneck 공정을 식별하고 부하를 분산한다 (공정별 가동률 ≤ 85% 유지 권장).
3. 납기 충돌 시 우선순위 결정: **납기 임박 → 전략 고객 → 수익성** 순.
4. OT·외주 필요 여부를 판단하여 {{dept_management}}의 승인을 득한다.
5. 확정된 계획을 **주간 생산 계획서(QF-PR01-01)**에 기재하고 관련 부서에 배포한다.

---

### P3. 자재 소요 계획(MRP) — {{dept_production}} + {{dept_purchasing}} (주간 계획 확정 후 1일 이내)

1. 완제품 BOM(자재 명세서)을 전개하여 필요 자재·부품 수량을 산출한다.
2. 각 자재의 **리드타임(발주~입고 기간)**을 반영하여 발주 시점을 역산한다.
3. 안전재고 미만 자재는 즉시 **구매 요청서(QF-PU01-01)**를 발행하여 {{dept_purchasing}}에 전달한다.
4. MRP 결과를 **MRP 소요 계획서**에 기록하고 주간 회의에서 공유한다.

---

### P4. 작업 지시 발행 — {{dept_production}} (전일 마감까지)

1. 익일 생산 계획에 따라 **작업 지시서(QF-PR01-02)**를 발행한다.
2. 작업 지시서 필수 기재 항목:
   - 품번·품명, 작업 수량, 작업 순서, 담당 라인·설비
   - 도면 번호, 작업 지침서 번호, 납기 일자
   - 특별 주의 사항(특수 공정, 고객 특별 특성 적용 여부)
3. 작업자에게 **도면·작업 지침서·Control Plan**을 함께 제공한다.

{{#if cert.iatf}}
> **IATF 요건**: 고객 특별 특성(SC/CC) 포함 제품 작업 시,
> 해당 **관리 계획서(Control Plan)** 적용 여부를 반드시 확인하고 작업 지시서에 명시한다.
{{/if}}

---

### P5. 생산 진도 관리 — {{dept_production}} (일간 3회: 오전·오후·마감)

1. 시간별·일별 생산 실적을 계획 대비 점검한다.
2. 계획 대비 달성률 ≥ 95% 유지를 목표로 하며, 이탈 시 즉시 원인을 파악하고 대응 조치를 취한다.
3. **라인 정지 기준**:
   - 품질 이상(치수 불량, 외관 불량 등) → **즉시 정지** + QP-QC01(공정 품질) 연계
   - 안전 이상(화재·부상 등) → **즉시 정지** + 안전 관리자 보고
4. 긴급 대응이 필요한 경우 OT 또는 라인 전환을 {{dept_management}}에 보고 후 결정한다.
5. 고객 납기 위험이 예상되면 **즉시** {{dept_sales}}에 통보하고 대응 방안을 협의한다.

---

### P6. 생산 실적 분석 — {{dept_production}} + {{dept_quality}} (주간·월간)

1. 생산 계획 대비 달성률, 불량률, 손실 분석을 **생산 실적 보고서(QF-PR01-03)**에 기록한다.
2. 주요 손실 유형을 분류한다: 설비 고장 / 자재 결품 / 품질 불량 / 인원 부족.
3. 반복 손실 항목에 대해 개선 목표를 설정하고 **QP-CA01(시정 조치)**와 연계한다.
4. 월간 결과를 {{dept_management}}에 보고하고 경영검토(QP-MR01) 입력 자료로 활용한다.

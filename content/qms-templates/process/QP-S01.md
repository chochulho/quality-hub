# 협력사 평가·관리 프로세스

**문서번호**: QP-S01  **버전**: {{rev_no}}  **발행일**: {{created_at}}
**회사명**: {{company_name}}  **프로세스 오너**: {{dept_purchasing}}

---

## 1. 목적

공급 품목의 품질·납기·기술 역량을 갖춘 협력사를 선정·등록·평가하여 공급망 전반의 품질을 확보하고, {{cert_target}} 외부 공급 프로세스 관리 요건을 충족한다.

## 2. 적용 범위

{{company_name}}의 승인 공급자 목록(AVL, Approved Vendor List)에 등재된 모든 협력사에 적용한다. 외주 가공 협력사는 QP-OUT1(외주 관리 프로세스)을 추가로 적용한다.

## 3. 책임과 권한

| 역할 | 책임 | 권한 |
|------|------|------|
| {{dept_purchasing}} | AVL 유지, 협력사 성과 종합 평가, 신규 등록 절차 주관, 계약 관리 | AVL 등재·제외 결정 (위원회 승인 후) |
| {{dept_quality}} | 협력사 현장 품질 감사, 품질 역량 평가, 수입 불합격 데이터 제공 | 협력사 품질 승인 / 퇴출 권고 |
| {{dept_engineering}} | 협력사 기술 역량 평가 지원, 도면·규격 요건 전달 | 기술 승인 및 공정 능력 기준 제시 |
| {{dept_management}} | 협력사 등록·퇴출 최종 승인 | 전략 협력사 지정 |

## 4. 프로세스 흐름 (SIPOC)

| 구분 | 내용 |
|------|------|
| **SUPPLIER** | 협력사 후보, 고객(CSL 요구 시), {{dept_purchasing}}, {{dept_quality}} |
| **INPUT** | 협력사 후보 목록, 평가 기준서(QF-S01-02), 고객 승인 요건(CSR), 거래 실적 데이터 |
| **PROCESS** | P1. 신규 협력사 등록 → P2. 거래 개시 → P3. 정기 성과 평가 → P4. 개선 조치 → P5. 협력사 퇴출 |
| **OUTPUT** | 승인 공급자 목록(AVL, QF-S01-03), 협력사 평가 보고서(QF-S01-04), 개선 조치 기록(QF-S01-05) |
| **CUSTOMER** | {{dept_production}}, {{dept_quality}}, 고객 |

## 5. 세부 절차

### P1. 신규 협력사 등록 — {{dept_purchasing}} + {{dept_quality}} (거래 개시 최소 4주 전)

**1단계: 서류 심사**
- 사업자등록증, ISO/IATF 인증 여부 및 유효성, 재무 현황(최근 2년 결산), 법적 규제 준수 현황 검토
- 공급자 자기 평가서(QF-S01-01) 제출 요구 및 검토

**2단계: 샘플 평가**
- 시제품 또는 샘플 제출 요구 → {{dept_quality}} 수입검사 기준 적용 검사
- 검사 결과 합격 시에만 3단계 진행

**3단계: 현장 심사** (QF-S01-02 현장 심사 체크리스트 사용)
- 공정 현황 및 생산 능력, 품질 시스템(문서 관리·내부 감사·시정조치) 운영 수준
- 4M(사람·설비·재료·방법) 관리 체계, 특별 특성 관리 능력, 계측기 교정 현황

**4단계: 등록 위원회 승인**
- {{dept_purchasing}} + {{dept_quality}} + {{dept_engineering}} 공동 검토 → {{dept_management}} 최종 승인
- AVL(QF-S01-03) 등재, 초기 등급 부여(신규: B등급 시작)

{{#if cert.iatf}}
> **IATF 16949 §8.4.1**: 고객 지정 공급자(CSL, Customer-Specified Source)는 고객 승인 없이 변경 불가하며 AVL에 CSL 표시를 별도 관리한다. 2nd tier 협력사 관리 요건이 고객 CSR에 포함된 경우 이를 1st tier 협력사에 전달하고 이행을 확인한다.
{{/if}}

### P2. 거래 개시 — {{dept_purchasing}} (AVL 등재 후)

1. **AVL 등재 및 조건 확정**: 공급 품목, 단가, 납기 기준, 품질 요건 협의
2. **구매 계약 체결**: 고객 CSR 전달 의무, 품질 보증 조항, 불합격 시 처리 조항(반품·비용 부담) 포함
3. **CSR 전달 확인**: 고객 특별 요건 수령·이해 확인서 보관 (IATF 소급 증빙용)
4. **초기 공급 모니터링**: 최초 5 Lot는 강화 수입검사 적용 (전수 또는 AQL 강화 등급)

### P3. 정기 성과 평가 — {{dept_purchasing}} + {{dept_quality}} (분기·연간)

**평가 항목 및 배점**:

| 평가 항목 | 배점 | 세부 기준 |
|---------|------|---------|
| 품질 (수입 불합격률·클레임 건수) | 40점 | 분기 수입 불합격률, 발생 클레임 건수·심각도 |
| 납기 준수율 | 30점 | 약정 납기 대비 실제 입고 일자 |
| 기술 지원·반응도 | 20점 | 시정조치 기한 준수율, 기술 문의 응답 속도 |
| QMS 운영 수준 | 10점 | 인증 유지 여부, 내부 감사 실적, 문서 관리 |

**등급 기준**:

| 등급 | 점수 | 의미 | 관리 방법 |
|------|------|------|---------|
| A등급 (우수) | 80점 이상 | 안정적 공급 | 연 1회 서면 평가 |
| B등급 (양호) | 60~79점 | 일부 개선 필요 | 반기 1회 리뷰 |
| C등급 (요주의) | 59점 이하 | 개선 필수 | P4 조치 즉시 시행 |

### P4. 개선 조치 — {{dept_purchasing}} + 협력사 (C등급 판정 후 즉시)

1. **개선 계획서 제출 요구**: C등급 판정 통보 후 30일 이내 개선 계획서 제출 요구 (QF-S01-05 양식 사용)
2. **현장 지도**: 필요 시 {{dept_quality}} + {{dept_engineering}} 현장 기술 지도 실시
3. **재평가**: 개선 계획서 이행 완료 후 60일 이내 재평가 실시
4. **조건부 공급 지속**: 재평가 B등급 이상 회복 시 조건부 거래 유지 (수입검사 강화 6개월 유지)
5. **재평가 재실패 시**: P5 퇴출 절차 진행

### P5. 협력사 퇴출 — {{dept_purchasing}} + {{dept_quality}} (등록 위원회 결정)

1. **퇴출 사유**: C등급 2회 연속, 중대 품질 사고(고객 클레임 직접 유발), 법적·윤리적 위반, 사업 중단
2. **위원회 심의**: {{dept_purchasing}} + {{dept_quality}} + {{dept_management}} 심의·승인
3. **AVL 제외 처리**: AVL에서 제외 표시, 제외 사유 및 일자 기록 보관 (5년)
4. **진행 중 발주 처리**: 퇴출 결정 후 신규 발주 즉시 중단, 기 발주 건은 완료 또는 이관
5. **대체 공급처 확보**: 퇴출 전 대체 공급처를 AVL에 등재 완료 후 퇴출 진행 (공급 리스크 최소화)

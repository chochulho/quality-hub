# 외주품 수입검사 프로세스

**문서번호**: QP-S02  **버전**: {{rev_no}}  **발행일**: {{created_at}}

**회사명**: {{company_name}}  **프로세스 오너**: {{dept_quality}}

---

## 1. 목적

외주 가공 완료품 및 구매 부품을 수령 시 품질 기준에 따라 검사하여 부적합 자재가 공정에 투입되지 않도록 한다.

## 2. 적용 범위

외주 가공품, 구매 부품, 원자재 등 외부에서 입고되는 모든 자재에 적용한다.

## 3. 책임과 권한

| 역할 | 책임 |
|------|------|
| {{dept_quality}} | 수입검사 기준 수립, 검사원 교육·관리 |
| 검사원 | 수입검사 실시, 기록 작성 |
| {{dept_production}} | 합격 자재 사용, 불합격 격리 협조 |
| {{dept_purchasing}} | 불합격 자재 공급자 반품·조치 |

## 4. 절차

### 4.1 검사 기준 등록

1. 각 품목별 **수입검사 기준서(QF-S02-01)** 작성:
   - 검사 항목 (치수, 외관, 재질, 기능)
   - 샘플링 계획 (AQL 기반 또는 전수)
   - 합격·불합격 기준
2. 특별특성(SC)은 반드시 포함하고 측정 방법 명시

### 4.2 입고 확인

1. 납품서와 발주서 대조 (품번, 수량, 개정 번호)
2. 포장 상태 확인 (파손, 오염, 라벨 여부)
3. 이상 시 즉시 {{dept_purchasing}}에 보고

### 4.3 수입검사 실시

| 검사 항목 | 방법 | 빈도 |
|---------|------|------|
| 외관 검사 | 육안, 확대경 | 전수 또는 샘플링 |
| 치수 검사 | 캘리퍼스, 마이크로미터 | 샘플링 (AQL 기준) |
| 특별특성 | 해당 계측기 | 전수 또는 강화 샘플링 |
| 재질·성능 | 시험 성적서 확인 또는 분석 | Lot당 1회 |

**샘플링 기준** (ANSI/ASQ Z1.4 기반):
- 일반 검사 수준 II, AQL = 1.0 (특별특성), AQL = 2.5 (일반)

### 4.4 판정 및 처리

| 판정 | 식별 | 처리 |
|------|------|------|
| **합격** | 초록 합격 태그 + 입고 스탬프 | 자재 창고 입고 |
| **조건부 합격** | 노랑 특채 태그 + 특채 번호 | 특채 처리 후 제한 사용 |
| **불합격** | 빨강 불합격 태그 | 격리 → 반품 또는 폐기 |

### 4.5 불합격 처리

1. 불합격 보고서(QF-S02-02) 작성 → {{dept_purchasing}}에 즉시 통보
2. {{dept_purchasing}}은 공급자에게 반품 또는 8D 요청
3. 반복 불합격 시 해당 공급자 등급 하향 (QP-S01 연계)

### 4.6 긴급 입고 (Skip-Lot)

생산 긴급 상황 시 절차:
1. {{dept_quality}} 팀장 승인 하에 잠정 사용 가능
2. 잠정 사용 자재는 명확히 식별하고 추적 관리
3. 검사 완료 전 출하 금지

## 5. KPI

| 지표 | 유형 | 산식 | 목표 | 주기 |
|------|------|------|------|------|
| 수입 불합격률 | 효과성 | 불합격 lot / 전체 입고 lot × 100 | ≤ 0.5% | 월간 |
| 검사 소요 시간 | 효율성 | 입고~판정 완료 평균 시간 | ≤ 4시간 | 주간 |
| 재발 불합격률 | 효과성 | 동일 원인 재불합격 / 전체 | ≤ 10% | 분기 |

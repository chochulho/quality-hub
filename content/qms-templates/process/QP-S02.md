# 외주품 수입검사 프로세스

**문서번호**: QP-S02  **버전**: {{rev_no}}  **발행일**: {{created_at}}
**회사명**: {{company_name}}  **프로세스 오너**: {{dept_quality}}

---

## 1. 목적

외부에서 입고되는 구매품·외주품에 대해 품질 기준에 따른 검사를 실시하여 부적합품이 생산 공정에 투입되지 않도록 방지하고, 공급망 품질 데이터를 체계적으로 축적한다.

## 2. 적용 범위

{{company_name}}에 입고되는 모든 외주 가공품, 구매 부품, 원자재에 적용한다. 단, 고객이 직접 지급하는 자재(고객 지급품)는 별도 절차에 따른다.

## 3. 책임과 권한

| 역할 | 책임 | 권한 |
|------|------|------|
| {{dept_quality}} | 수입검사 기준 수립·유지, 검사원 교육·관리, 검사 판정 최종 확정 | 불합격 격리 명령, 특채 승인(팀장 이상) |
| 검사원 | 수입검사 실시, 성적서 작성, 이상 즉시 보고 | 입고 현장 접근 및 샘플 채취 |

| 구분 | 내용 |

| **PROCESS** | P1. 입고 접수 → P2. 외관·수량 검사 → P3. 치수·성능·재질 검사 → P4. 합격/불합격 처리 → P5. 기록 관리 |
| **OUTPUT** | 수입검사 성적서(QF-S02-02), 합격 태그, 불합격 보고서(QF-S02-03), 반품 지시서 |
| **CUSTOMER** | {{dept_production}}, {{dept_quality}}, 공급자 (피드백) |

## 5. 세부 절차

### P1. 입고 접수 — {{dept_purchasing}} → {{dept_quality}} (입고 당일)

1. **발주서 대조**: 품번, 수량, 납기, 도면 개정 Rev., 공급자 성적서(CoC) 동봉 여부 확인
2. **검사 대기 구역 격리**: 검사 완료 전 생산 투입 금지 — 검사 대기 태그(황색) 부착 및 격리 구역 이동
3. **검사 기준 확인**: 해당 품목의 수입검사 기준서(QF-S02-01) 존재 여부 확인 → 없는 경우 {{dept_quality}} 팀장에 즉시 보고
4. **긴급 납기 대응 (조건부 불출)**: 생산 긴급 상황 시 {{dept_quality}} 팀장 이상 서면 승인 후 조건부 불출 가능 → 검사 완료 전 최종 출하 금지, 사용 이력 추적 관리

### P2. 외관·수량 검사 — {{dept_quality}} (접수 후 1영업일 이내)

1. **외관 검사**: 육안 및 확대경으로 표면 손상, 오염, 녹, 변형 여부 확인
2. **수량 검사**: 납품 수량과 발주서 수량 대조, 로트번호·제조일자 라벨 확인
3. **포장 상태 확인**: 포장 파손, 충격 지시기(Shockwatch) 발동 여부 확인
4. **AQL 샘플 수량 결정** (QF-S02-01 샘플링 계획 기준):
   - 적용 표준: ANSI/ASQ Z1.4 (계수형 샘플링)
   - 검사 수준: 일반 검사 수준 II
   - AQL: 특별 특성 포함 품목 AQL 1.0, 일반 품목 AQL 2.5

### P3. 치수·성능·재질 검사 — {{dept_quality}} (1~2영업일)

1. **치수 검사**: 도면 치수 대비 캘리퍼스·마이크로미터·CMM 등 적합한 계측기로 측정, 성적서(QF-S02-02) 기록
2. **성능·기능 시험**: 해당 품목의 기능 요건에 따른 조립 적합성, 토크, 인장 강도 등 시험
3. **재질 확인**: 공급자 성적서(Mill Sheet, CoC) 검토 → 필요 시 현장 성분 분석 또는 외부 기관 의뢰
4. **공급자 성적서(CoC) 검토**: 치수·성능 값이 규격 내 있는지 확인, 서명·인감 유효성 검토

{{#if cert.iatf}}
> **IATF 16949 §8.4.3**: 고객 특별 특성(SC/CC)이 포함된 구매 부품은 전수 검사 또는 강화 샘플링(AQL 0.65 이하)을 적용하며, 합격 기록을 고객 요구 시 제출할 수 있도록 보관한다.
{{/if}}

### P4. 합격/불합격 처리 — {{dept_quality}}

**판정 기준 및 처리 방법**:

| 판정 | 식별 태그 | 처리 |
|------|---------|------|
| **합격** | 초록 합격 태그 (검사일·검사원·성적서 번호 기재) | 창고 이관 승인, 입고 스탬프 날인 |
| **특채 (긴급 시)** | 노랑 특채 태그 (특채 번호·사용 제한 조건 기재) | {{dept_quality}} 팀장 이상 서면 승인 필수, 사용처 및 주의사항 명시, 이력 추적 |
| **불합격** | 빨강 불합격 태그 (불합격 사유 기재) | 격리 보관 구역 즉시 이동, P4-1 불합격 처리 절차 실행 |

**P4-1. 불합격 처리 세부 절차**:
1. 불합격 보고서 작성 (QF-S02-03): 품번, Lot번호, 불합격 수량, 불합격 사유, 검사원 서명
2. {{dept_purchasing}}에 즉시 통보 → 공급자에 반품 지시 또는 8D 시정조치 요구 (30일 이내)
3. QP-CA01 시정조치 프로세스 연계: 반복 불합격(동일 공급자·동일 원인 2회 이상) 시 필수
4. 협력사 등급 영향 반영: QP-S01 협력사 평가 데이터에 불합격 이력 반영
5. **반품 지시**: 확정 반품 Lot는 {{dept_purchasing}} 통해 공급자에 반품 지시 및 비용 청구 협의

### P5. 기록 관리 — {{dept_quality}} (3년 보관)

1. **수입검사 성적서 보관** (QF-S02-02): 품번·Lot별 검사 결과, 판정, 검사원 서명, 계측기 번호 기재 → 3년 보관
2. **불합격 이력 관리** (QF-S02-03): 불합격 사유, 조치 내용, 재발 여부 추적 → 공급자별 이력 통계 관리
3. **특채 기록 관리**: 특채 승인 문서, 사용 이력, 추적 가능성 기록 → 3년 보관
4. **월간 수입검사 통계 보고**: 입고 Lot 수, 불합격 Lot 수, 불합격률, 주요 불합격 사유 분석 → {{dept_management}} 보고

## 1. 목적

내부·외부 부적합의 근본 원인을 제거하여 재발을 방지하고, QMS의 지속적 개선을 추진한다.

## 2. 적용 범위

{{company_name}}에서 발생하는 모든 부적합 — 내부검사 부적합, 고객 클레임, 내부심사 NCR, 공급자 부적합 — 에 적용한다.

## 3. 책임과 권한

| 역할 | 책임 |
|------|------|
| {{dept_quality}} | 부적합 등록, 원인 분석 주도, 효과 검증 |
| 해당 부서장 | 조치 계획 수립 및 실행 |
| {{dept_management}} | 중대 부적합 자원 지원 승인 |

## 4. 절차

### 4.1 부적합 등록

1. 부적합 발생 시 NCR(부적합 보고서, QF-CA01-01)을 작성하고 {{dept_quality}}에 통보
2. NCR 번호 체계: `NCR-YYYY-NNN` (예: NCR-2026-001)
3. 심각도 분류: Critical(즉시) / Major(7일) / Minor(30일)

### 4.2 즉각 조치 (Containment)

1. 부적합품 격리 및 식별 표기 (빨간 태그)
2. 영향 범위 파악: 사내 재고, 공정 중, 고객 인도 물량
3. 고객 통보 필요 시 QP-C01 절차 병행

### 4.3 근본 원인 분석

- **5-Why**: 반복성 부적합에 효과적
- **특성요인도**: 복합 원인 파악
- **Is/Is-Not 분석**: 문제 경계 명확화
- PFMEA 연계: 기존 관리 수단 누락 여부 확인

### 4.4 시정 조치 수립 및 실행

1. 근본 원인별 시정 조치 계획 수립 (담당자·완료일 지정)
2. 유사 공정/제품 수평 전개 여부 결정
3. 문서 갱신: PFMEA, Control Plan, 작업지침서

### 4.5 효과 검증

1. 조치 완료 후 **최소 3개월** 실데이터로 효과 확인
2. 재발 없음 확인 후 NCR 완결 처리
3. 효과 미확인 시 → 4.3 단계로 재개

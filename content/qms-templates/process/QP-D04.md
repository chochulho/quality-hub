## 1. 목적

고객 도면·사양을 입력으로 제조 공정을 체계적으로 설계하고, PFMEA를 통해 공정 리스크를 사전에 제거한다.

## 2. 적용 범위

고객 도면 접수 후 신규 공정 구성, 양산 이관, 공정 주요 변경 시 적용한다.

## 3. 절차

### 3.1 고객 도면·사양 접수 및 검토

1. {{dept_sales}}로부터 도면·사양·CSR을 수령
2. 제조 가능성 검토 (Feasibility):
   - 치수·공차 달성 가능 여부
   - 설비·지그 보유 여부
   - 특별특성 관리 방법
3. 불명확한 사항은 고객에게 서면 질의

### 3.2 공정 흐름도 작성

1. 원자재 입고 → 각 가공 단계 → 검사 → 출하까지 전 흐름 도식화
2. 각 단계에 공정 번호(10, 20, 30…) 부여
3. 특별특성이 발생하는 공정 단계 표시

### 3.3 PFMEA 작성

AIAG-VDA 7단계 방법론 적용 (QP-D03 참조):
1. 각 공정 단계별 고장 모드·영향·원인 분석
2. S/O/D 평가 및 AP(Action Priority) 판정
3. AP=H 항목 즉시 예방·감지 조치 수립

### 3.4 Control Plan 작성

| 항목 | 내용 |
|------|------|
| 공정 번호 | 공정 흐름도와 동일 번호 |
| 제품·공정 특성 | 관리 대상 치수·변수 |
| 특별특성 표기 | △(고객 SC), ◇(내부 SC) |
| 관리 방법 | SPC / 전수 검사 / 순번 검사 |
| 반응 계획 | 이상 발생 시 대응 절차 |

### 3.5 작업지침서(WI) 작성

- Control Plan의 각 공정 단계별 작업지침서 작성
- 그림·사진 활용하여 작업자가 이해하기 쉽게 작성
- 특별특성 작업 단계는 별도 강조 표기

### 3.6 양산 이관 (Run-at-Rate)

1. 실제 양산 조건에서 시험 생산 (최소 300 pcs 또는 1시간)
2. 공정능력 확인: 특별특성 Cpk ≥ 1.67
3. PPAP 패키지 작성 (고객 제출 레벨에 따라)

## 4. KPI

| 지표 | 유형 | 산식 | 목표 | 주기 |
|------|------|------|------|------|
| 공정 설계 일정 준수율 | 효과성 | 일정 내 완료 / 전체 × 100 | ≥ 90% | 프로젝트별 |
| PFMEA AP=H 조치율 | 효과성 | 조치 완료 H / 전체 H × 100 | 100% | 월간 |
| 초기 공정능력 달성율 | 효율성 | Cpk ≥ 1.67 달성 SC 수 / 전체 SC × 100 | ≥ 90% | 프로젝트별 |

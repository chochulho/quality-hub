## 1. 목적

제품 설계를 동반하지 않는 신규 공정 설계 또는 기존 공정 변경 시, PFMEA와 Control Plan을 체계적으로 갱신하여 공정 품질 리스크를 사전에 제거하고, 1급(고객 통보 필요) 변경에 대해서는 고객의 사전 승인을 획득한다.

---

## 2. 적용 범위

다음 상황에서 본 절차를 적용한다.

| 적용 사유 | 변경 등급 판정 | 비고 |
|---|---|---|
| 신규 공정 라인 구축 또는 신규 제품 수주 (공정 설계 포함) | 1급 | APQP Phase 3 연계 (QP-D02) |
| 핵심 공정 설비 교체 또는 신규 도입 | 1급 또는 2급 | 고객 CSR 기준 우선 |
| 공정 파라미터·조건 변경 (온도·압력·속도 등) | 1급 또는 2급 | 특별 특성 영향 여부로 구분 |
| 협력사(외주 공정) 변경 | 1급 | 고객 승인 필수 |
| 원자재·보조 재료 변경 | 1급 또는 2급 | 재질 성적서 갱신 |
| 제조 공장 또는 라인 이설 | 1급 | PPAP 갱신 필요 |
| 내부 공정 조정 (일상적 미세 조정) | 3급 | 내부 기록만 |

## 5. 세부 절차

### P1. 공정 변경 검토 및 등급 분류

**담당**: {{dept_engineering}} + {{dept_quality}}  **기한**: 변경 요청 수령 후 3영업일 이내

1. **공정 변경 요청서(QF-D04-00)** 작성: 변경 내용, 변경 사유, 예상 적용일
2. **변경 등급 분류** (고객 CSR 우선, 미명시 시 아래 기준 적용):

| 등급 | 기준 | 처리 |
|---|---|---|
| **1급** (고객 통보·승인 필요) | 특별 특성 영향, 설비 교체, 협력사 변경, 공장 이설, 재질 변경 | 고객 승인 후 적용 (P5 필수) |
| **2급** (내부 승인) | 일반 특성 관련 공정 파라미터 변경, 비핵심 설비 교체 | {{dept_management}} 승인 후 적용 |
| **3급** (일상적 조정) | 관리 한계 내 미세 조정, 동일 규격의 소모품 교체 | 담당자 기록 후 즉시 적용 |

3. **고객 특별 특성(SC/CC) 영향 여부 판단**: 영향이 있으면 반드시 1급으로 상향
4. 1급 변경의 경우: **고객 사전 승인 요청** (P5 착수), 승인 수령 전 변경 적용 금지

---

### P2. PFMEA 갱신

**담당**: 다기능 팀 ({{dept_engineering}} 주관)  **기한**: 변경 승인(내부) 후 7영업일 이내

1. 변경된 공정 단계 식별 → 해당 PFMEA 행 재검토
2. **AIAG-VDA 4th Edition 7단계 방법론 적용** (QP-D03 상세 참조):
   - 구조 분석: 변경 공정 요소(4M) 반영
   - 기능·고장 분석: 변경으로 인한 신규 고장 모드·원인 도출
   - 위험 분석: S·O·D 재평가 → AP(Action Priority) 재산정
3. **AP-H 항목**: 즉시 예방·감지 조치 수립 및 완료일 지정
4. PFMEA 개정 이력 기록 (변경 사유·변경 내용·담당자·날짜)
5. {{dept_quality}} 최종 검토 및 승인

---

### P3. Control Plan 갱신

**담당**: {{dept_quality}} + {{dept_production}}  **기한**: PFMEA 갱신과 동시 (≤7일)

1. 변경 공정 단계의 **공정 특성·제품 특성·관리 방법·반응 계획** 갱신
2. **특별 특성(SC/CC) 관리 방법 확인**: SPC 또는 100% 검사 적용 여부 재검토
3. Control Plan 3단계 갱신 필요 여부 확인:
   - Production Control Plan: 반드시 갱신
   - Pre-launch Control Plan: 신규 시험 생산 수반 시 갱신
4. **작업지침서(WI) 갱신 지시**: 변경 공정 단계의 WI → {{dept_production}}에 갱신 지시 및 교육 실시
5. Control Plan 개정 이력 기록 → {{dept_quality}} 승인 후 배포

---

### P4. 시험 생산 및 공정 능력 확인

**담당**: {{dept_production}} + {{dept_quality}}  **기한**: 변경 적용 전 (내부 승인 후)

1. **소규모 시험 생산(Trial Run)** 실시:
   - 변경된 조건 적용, 최소 100 pcs 또는 연속 30분 이상 (규모는 변경 등급에 비례)
   - 치수·기능·외관 전수 또는 샘플 검사
2. **공정 능력 확인**:
   - 특별 특성: **Cpk ≥ 1.67** (초기 양산 기준, IATF §8.1 요건)
   - 일반 특성: **Cpk ≥ 1.33**
   - 미달성 시 → 공정 조건 재조정 → 재측정, PFMEA 추가 조치
3. **측정 시스템(계측기) 적합성 확인**: 변경 공정에 사용되는 계측기 Gage R&R 유효성 검토
4. 시험 생산 결과 보고서(QF-D04-02) 작성 → {{dept_quality}} 승인

---

### P5. 고객 통보 및 승인 (1급 변경 해당)

**담당**: {{dept_quality}} + {{dept_sales}}  **기한**: 내부 시험 생산 완료 후

1. **공정 변경 통보서(QF-D04-01)** 작성:
   - 변경 내용, 변경 사유, 적용 예정일, 시험 생산 결과 요약 포함
2. 통보서 + 관련 자료(PFMEA 갱신본, Cpk 결과, Control Plan 갱신본) 고객에게 제출
3. **고객 승인 수령 후 변경 적용**: 고객 구두 승인은 인정하지 않음 (서면·이메일·시스템 승인)
4. 고객 요구 시 **PPAP 갱신 제출**:
   - 고객 지정 Level에 따른 PPAP 요소 재제출
   - PSW(Part Submission Warrant) 갱신 발행
5. 승인 기록 보관: QP-DC01 기준에 따라

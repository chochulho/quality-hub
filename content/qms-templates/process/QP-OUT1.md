## 1. 목적

외주 공정 및 위탁 작업에 대해 품질·납기·비용을 체계적으로 관리하고, 외주업체의 품질 역량을 확보하여 외주 완료품이 내부 생산품과 동일한 품질 기준을 충족하도록 한다.

## 2. 적용 범위

{{company_name}}이 외부 업체에 위탁하는 모든 공정(가공, 도금, 도장, 열처리, 용접, 조립 등)과 이를 위해 구매하는 외주 서비스에 적용한다.

## 5. 세부 절차

### P1. 외주업체 등록 및 평가 — {{dept_purchasing}} + {{dept_quality}} (신규 시 최소 4주)

1. **서류 심사**: 사업자등록증, ISO/IATF 인증 여부, 재무 현황, 법적 규제 준수 현황 검토
2. **현장 심사**: 공정 현황, 품질 시스템, 4M 관리 수준, 특별 특성 관리 능력 평가 (QF-OUT1-01 현장 심사 체크리스트 사용)
3. **시제품 평가**: 시범 발주 → {{dept_quality}} 수입검사 합격 여부 확인
4. **등록 심사위원회 승인**: {{dept_purchasing}} + {{dept_quality}} + {{dept_engineering}} 공동 승인 → AVL(승인 공급자 목록) 등재
5. **기존 외주업체**: 연간 재평가 실시 (P5 참조), 미달 시 조건부 유지 또는 퇴출

{{#if cert.iatf}}
> **IATF 16949 §8.4.1**: 외주업체가 고객 지정 공급자(CSL)인 경우 고객 승인 절차를 별도 준수한다. 2nd tier 관리 요건이 고객 CSR에 포함된 경우 해당 요건을 외주업체에 명문화하여 전달한다.
{{/if}}

### P2. 외주 작업 발주 — {{dept_purchasing}} (요청 후 2영업일 이내)

1. **작업지시서 발행**: QF-OUT1-02 외주 작업 지시서에 품번, 수량, 납기, 작업 규격, 도면 번호, 개정 Rev. 명시
2. **기술 문서 전달**: 최신 도면, 작업표준서, 검사 기준, 특별 특성(SC/CC) 식별 목록 동봉
3. **특별 관리 항목 명시**: 고객 특별 특성이 포함된 공정은 별도 QC 계획 및 관리 방법 첨부
4. **발주 확인서 수령**: 외주업체로부터 수령·이해 확인 서명 보관 (CSR 전달 증빙용)

{{#if cert.iatf}}
> **IATF 16949 §8.4.3**: 고객 CSR을 공급망 전달 시 외주업체가 요건을 인지·준수함을 확인하는 서면 기록을 유지한다.
{{/if}}

### P3. 외주 진도 및 품질 관리 — {{dept_production}} + {{dept_quality}} (작업 진행 중)

1. **중간 확인**: 납기 리스크 또는 품질 이슈 발생 시 현장 방문 또는 사진·기록 제출 요구
2. **공정 감사**: 연 1회 이상 외주업체 현장 방문 심사 (고위험 외주는 분기 1회)
3. **Control Plan 적용 확인**: 외주업체가 자체 CP(Control Plan)를 보유하고 운영하는지 확인
4. **이슈 발생 시**: 작업 중단 지시 → 원인 파악 → 조건부 재개 또는 물량 전환

{{#if cert.iatf}}
> **IATF 16949 §8.4.2**: 외주 공정에 적용되는 Control Plan을 외주업체에 제공하거나, 외주업체 자체 CP가 당사 요건을 충족하는지 검토·승인한다.
{{/if}}

### P4. 수입검사 — {{dept_quality}} (입고 당일 ~ 다음 영업일)

1. **입고 접수**: 발주서와 대조(품번·수량·납기·개정 Rev.), 검사 대기 구역 격리 및 식별 태그 부착
2. **검사 기준 적용**: QF-OUT1-03 수입검사 성적서에 따라 AQL 샘플링 또는 전수 검사 실시
3. **합격 처리**: 합격 태그 부착 → 창고 이관 승인
4. **불합격 처리**: 불합격 태그 부착 → 격리 보관 → 부적합 보고서 발행(QP-CA01 연계) → 외주업체 시정조치 요구(8D 보고서, 30일 이내)
5. **반품 지시**: {{dept_purchasing}}을 통해 외주업체에 반품 지시 및 비용 청구 협의
6. **특채(긴급 사용 시)**: {{dept_quality}} 팀장 이상 서면 승인 필수, 사용처 및 주의사항 명시

### P5. 외주업체 성과 평가 — {{dept_purchasing}} + {{dept_quality}} (분기)

1. **평가 항목 및 배점**:

| 평가 항목 | 배점 | 세부 기준 |
|---------|------|---------|
| 품질 (불합격률·클레임) | 40점 | 분기 수입 불합격률, 발생 클레임 건수·심각도 |
| 납기 준수율 | 30점 | 약정 납기 대비 실제 입고일 |
| 기술 지원·반응도 | 20점 | 시정조치 기한 준수, 기술 문의 응답 속도 |
| QMS 운영 수준 | 10점 | 인증 유지, 내부 감사 실적, 문서 관리 |

2. **등급 분류**: A등급(90점 이상) / B등급(70~89점) / C등급(69점 이하)
3. **C등급 조치**: 개선 계획서 제출 요구(30일) → 현장 지도 → 60일 후 재평가 → 미달 시 퇴출 절차
4. **퇴출 처리**: AVL 제외 → 진행 중 발주 완료 또는 이관 → 대체 외주업체 긴급 확보

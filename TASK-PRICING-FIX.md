## 🔥 즉시 수정 (이번 세션)

### 1. 가격 페이지 — 정가/할인가/연납 표기 통일

**위치:** `app/(marketing)/pricing/page.tsx` (또는 해당 경로)

**현재 문제:**
- 베타 50% 할인가만 노출, 정가가 보이지 않음 → 50% 할인이라는 약속의 신뢰가 떨어지고, 베타 종료 시 충격 가격 변동.
- 연납 가격(예: ₩245,000)이 "/월"로 표기되어 "월 ₩245,000"으로 오인 가능. **결정적 표기 오류**.

**수정 후 표기 (모든 유료 플랜 카드 공통 패턴):**

```
Starter
도구 1개 선택

₩24,500 /월         ← 큰 글씨 (현재 표시 가격, 베타 50% 할인가)
정가 ₩49,000/월     ← 작은 글씨, 취소선 또는 회색
[베타 50% 할인 적용 중]  ← 배지

연간 결제 선택 시:
₩245,000/년 (월 ₩20,417 효과)
정가 ₩490,000/년
[2개월 무료]
```

**Team / Business도 동일 패턴:**
- Team: 베타 ₩74,500/월 / 정가 ₩149,000/월 / 연납 ₩745,000/년
- Business: 베타 ₩195,000/월 / 정가 ₩390,000/월 / 연납 ₩1,950,000/년

**가격 데이터 소스:** DB의 `plans` 테이블 (`price_krw_monthly`, `price_krw_yearly`). 하드코딩 금지.
- 정가 = `price_krw_monthly`
- 베타 할인가 = `price_krw_monthly * 0.5`
- 연납 = `price_krw_yearly`
- 베타 종료 후에는 자동으로 할인 배지가 사라지도록 환경변수 또는 DB 플래그로 제어 (예: `BETA_DISCOUNT_ACTIVE=true`).

### 2. 연납 토글 문구 수정

**위치:** 가격 페이지 상단 월간/연간 결제 토글

**현재:** "연간 결제 **10% 절약**"

**문제:** 실제 계산은 월 × 10 = 약 **16.7% 절약** (2개월 무료). 10%로 적으면 실제 혜택을 적게 보여줘 손해.

**수정 후:** "연간 결제 **2개월 무료** (약 17% 절약)"

### 3. 기능 비교 표 렌더링 수정

**위치:** 가격 페이지 "기능 비교" 섹션

**현재 문제:** 표의 셀이 비어 있음. 체크마크(✓)나 대시(—)가 표시되지 않음.

**원인 가능성:**
- 마크다운 테이블 셀 안에 빈 문자열이 들어가 있음
- 또는 컴포넌트 렌더링에서 boolean → 시각 요소 변환 누락

**수정:** 각 셀에 명시적 표시:
- 지원됨: ✓ (또는 체크 아이콘)
- 지원 안 됨: — 또는 ✕
- 옵션/애드온: 별도 표기 ("애드온", "선택")

**예시 표 (수정 후):**

| 기능 | Free | Starter | Team | Business |
|---|---|---|---|---|
| 품질 학습 위키 | ✓ | ✓ | ✓ | ✓ |
| SPC · QC7 계산 도구 | ✓ | ✓ | ✓ | ✓ |
| FMEA 체험 데모 | ✓ | ✓ | ✓ | ✓ |
| SaaS 도구 선택 | — | 1개 | 3개 | 전체 5개 |
| 팀원 수 | 3명 | 10명 | 30명 | 80명 |
| 사업장 | 1개 | 1개 | 2개 | 3개 |
| SSO 자동 로그인 | — | — | — | 애드온 |
| 이메일 지원 | — | ✓ | ✓ | ✓ |
| 채팅 지원 | — | — | ✓ | ✓ |

### 4. FMEA 데모 상태 표기 통일

**현재 모순:**
- 랜딩(`/`): "AI와 대화하면서 FMEA 자동 작성. 3분 체험. → 데모 시작하기" (작동 중처럼 표시)
- 데모 페이지(`/calculators/fmea-demo`): 시나리오 선택 UI 노출 (작동 일부)
- 가격 페이지(`/pricing`) Free 플랜: "**FMEA 체험 데모 (준비 중)**" 표기

**수정 방향 — 사용자 결정 필요:**
- (A) 데모가 실제 작동한다면 → 가격 페이지의 "(준비 중)" 표기 제거.
- (B) 데모가 아직 미완성이라면 → 랜딩에서도 "Beta", "준비 중" 같은 표기 일관 적용, 데모 페이지에는 "체험 일부 기능 제한" 안내.

**조처:** 현재 데모 페이지의 실제 작동 범위를 확인한 후, 사용자에게 어느 방향으로 갈지 묻고 통일.

### 5. "즉시 운영" 카피 톤 다운

**위치:** 랜딩 중반 "별도 시스템 구축 불필요" 섹션

**현재:** "기업회원 가입만으로 QMS 문서·교육·심사를 **즉시 운영합니다**."

**문제:** QMS는 회사 데이터 입력·설정·기존 문서 마이그레이션이 필요. "즉시 운영"은 첫 사용 후 실망으로 이어질 과장 약속.

**수정 후 옵션 (사용자 선택):**
- (A) "기업회원 가입만으로 **1주일 안에** QMS를 가동합니다."
- (B) "기업회원 가입 후 **즉시 도구를 사용**합니다. 별도 IT 구축 없이."
- (C) "**별도 IT 구축 없이** 가입 후 바로 QMS 운영을 시작합니다."
